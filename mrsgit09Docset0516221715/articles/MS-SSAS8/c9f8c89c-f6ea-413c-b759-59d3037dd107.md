<p>The syntax of <b>READ_CUBE_DIMENSION_ARRAY</b> is defined as
follows.           </p>

<dl>
<dd>
<div><pre> READ_CUBE_DIMENSION_ARRAY=
   OPEN(281)
   INT32(282)
   *DIMENSION_SECURITY
   CLOSE
</pre></div>
</dd></dl>

<table>
 <thead>
  <tr>
   <th>
   <p>Element</p>
   </th>
   <th>
   <p>Description</p>
   </th>
  </tr>
 </thead>
 <tr>
  <td>
  <p>OPEN(281)</p>
  </td>
  <td>
  <p>A common data structure that indicates the beginning
  of the element.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>INT32(282)</p>
  </td>
  <td>
  <p>A value that specifies the count of read-security
  dimension roles that are defined in the <a href="c527450b-f5bd-424b-8c98-ba6365288f35.htm#gt_a0c8d97b-322c-4117-8525-37e5f26751e7">cube</a>.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>DIMENSION_SECURITY</p>
  </td>
  <td>
  <p>A data structure that specifies dimension security for
  a specific dimension. This structure is repeated INT32(282) times.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>CLOSE</p>
  </td>
  <td>
  <p>A common data structure that indicates the end of the
  element.</p>
  </td>
 </tr>
