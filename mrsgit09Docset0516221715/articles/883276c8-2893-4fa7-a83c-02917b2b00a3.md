            <h1 class="heading">2.6.3.2.8 Moving a Message Object</h1>
        </div>
        <div id="mainSection">
            <div id="mainBody">
                <div id="allHistory" class="saveHistory"></div>
                <div id="sectionSection0" class="section" name="collapseableSection">
                    

<p>Moving a <a href="08220cc9-69b1-4072-a2e7-2a0ff201d505.htm#gt_b6c15d0c-d992-421d-ba96-99d3b63894cf">Message
object</a> from its parent <a href="08220cc9-69b1-4072-a2e7-2a0ff201d505.htm#gt_0682daa7-c1b8-419b-8a32-6048833d0b72">Folder
object</a> to another Folder object. This involves deleting the Message object
row from the old parent's Contents TC (section <a href="5a0450b5-61c3-4bb0-9837-fd14a00040d2.htm">2.6.2.4.7</a>), and adding it
to the new parent's Content TC (section <a href="1a94f596-d840-4f66-824e-af1024fb6944.htm">2.6.2.4.2</a>). Also, some
properties of both Folder object PCs (for example, message count) need to be
updated (section <a href="d17234d1-4de9-436e-a412-186b42dd1a8b.htm">2.6.3.2.2</a>).</p>
