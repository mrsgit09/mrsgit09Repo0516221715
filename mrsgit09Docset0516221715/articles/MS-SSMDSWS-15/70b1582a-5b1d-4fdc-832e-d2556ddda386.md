<p><b>AccessPermission</b>: <a id="OLE_LINK98"></a><a id="OLE_LINK97">Specifies </a>the type of access, if the user has been
granted access permission.</p>

<p><b>Attributes</b>: Specifies the attributes of the
member.</p>

<p><b>AuditInfo</b>: Specifies the modification history
of the member.</p>

<p><b>Collections</b>: Specifies the collection in which
this member is included.</p>

<p><b>DerivedHierarchyParentIds</b>: Specifies the
identifiers of the derived hierarchy parents of the members. This element is
used only when the get criteria specify a derived hierarchy level and request
membership information.</p>

<p><b>IsSoftDeleted</b>: Specifies whether the member is
deactivated.</p>

<p><b>MemberId</b>: Specifies the name, code, and type
of the member.</p>

<p><b>Parents</b>: Specifies the parents of the member.</p>

<p><b>SecurityPermission</b>: Specifies the
authorization of the user for the member.</p>

<p><b>TransactionAnnotation</b>: Specifies the
annotation that is applicable to this member's transaction during creation.</p>

<p><b>ValidationStatus</b>: Specifies the validation status of
the member.</p>
