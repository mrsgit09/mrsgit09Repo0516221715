<p>Security is opaque to file systems. Some file systems store
security descriptors as opaque blobs and then call security support routines to
perform the necessary security checks. Other file systems do not implement
security. Security considerations are called out in the sections where they are
used. Please refer to <mshelp:link keywords="953d700a-57cb-4cf7-b0c3-a64f34581cc9" tabindex="0">[MS-AUTHSOD]</mshelp:link>
for a security overview.</p>
