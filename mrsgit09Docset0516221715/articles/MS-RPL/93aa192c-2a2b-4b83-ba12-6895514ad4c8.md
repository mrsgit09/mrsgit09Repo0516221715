<p>The <b>FontFamily</b> property specifies the name of the
font family for a contiguous set of characters. If this property is specified,
it MUST be part of a <a href="04bf25a1-2f43-4acf-b9eb-b9fa2dc45202.html">Style</a>
record. If the <b>FontFamily</b> property is not specified, the default value
is used.</p>
