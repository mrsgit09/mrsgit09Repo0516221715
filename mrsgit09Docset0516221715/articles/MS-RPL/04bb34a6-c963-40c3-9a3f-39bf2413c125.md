<p><b>useShared (1 byte): </b>A byte that specifies the
start of the <b>UseSharedImageDataProperties</b> record. The value of the <b>useShared</b>
field MUST be 0x02.</p>

<p><b>offset (8 bytes): </b>A signed integer that
specifies the start of the <a href="aaead105-cdad-4c4e-a8d5-0b48c8b0e882.htm">ImageDataProperties</a>
record in the RPL stream; the <b>ImageDataProperties</b> record MUST have the <a href="1b93acb6-ccb8-494f-abe9-797e9d3ab199.htm">InlineSharedImageDataProperties</a>
record.</p>
