<a href="75ae48f7-746b-4b41-919c-6699fa28b3ef.htm#gt_c6f8e999-fca9-4e79-96e7-fb4c2c43d601">report item</a> that is
contained within a rectangle. If the <b>LinkToChild</b> property is specified,
it MUST be part of the <a href="23d76278-cee5-45ee-a361-a9d94d6d3300.htm">Rectangle.ElementProperties.InlineSharedElementProperties</a>
record.           </p>

<p><b>linkToChildStart (1 byte): </b>A byte that
specifies the start of the <b>LinkToChild</b> property. The value of the <b>linkToChildStart</b>
field MUST be the following value.</p>

<dd>

  </td>
  <td>
  <p>ElementProperties.InlineSharedElementProperties of <a href="e3de22f3-c3a5-4162-9e91-df6085f6fba0.htm">Rectangle</a></p>

<p><b>linkToChildValue (variable): </b>A <b>String</b>
variable that specifies the <b>ID</b> of a child report item contained directly
within a <b>Rectangle</b> that is the target location for the <a href="75ae48f7-746b-4b41-919c-6699fa28b3ef.htm#gt_46519c7a-96f3-40c7-9883-a17c87288632">document map</a> label (if
any).</p>
