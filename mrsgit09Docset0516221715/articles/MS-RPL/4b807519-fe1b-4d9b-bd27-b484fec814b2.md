<p>The <b>LeftIndent</b> property specifies the indentation for
all text within a paragraph from the left edge of the text box containing the <a href="3024abc3-23db-494b-a63a-6bd565e4500b.htm">Paragraph</a> record. If this
property is specified, it MUST be part of a <a href="d7f6cef2-01c6-4562-a4a0-5f205d79963e.htm">Paragraph.ElementProperties</a>
record.</p>

the text box containing the paragraph.</p>
