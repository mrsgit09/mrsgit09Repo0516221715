            <h1 class="heading">2.3.19 PageWidth</h1>
        </div>
        <div id="mainSection">

<p>The <b>PageWidth</b> property specifies the default width
for rendering a report in a <a href="75ae48f7-746b-4b41-919c-6699fa28b3ef.htm#gt_4d888382-2a0a-42b6-b95b-6a69e33360bc">physical-page
oriented renderer</a>. If the <b>PageWidth</b> property is specified, it MUST
be part of a <a href="1b8ce294-70e7-460a-b77c-96be5c46cef1.htm">PageLayout</a>
or <a href="0b56e16b-0d77-4cad-83a4-1ba0c046a35c.htm">PageProperties</a>
record. If the <b>PageWidth</b> property is not specified, the default value is
used.           </p>

<p><b>pageWidthStart (1 byte): </b>A byte that specifies
the start of the <b>PageWidth</b> property. The value of the <b>pageWidthStart</b>
field MUST be the following value.</p>

<dl>
<dd>
<table>
 <thead>

 </thead>
 <tr>
  <td>
  <p>0x11</p>
  </td>
  <td>
  <p>PageLayout</p>
  <p>PageProperties</p>
  </td>
 </tr>
</table>
</dd></dl>

<p><b>pageWidthValue (4 bytes): </b>A floating point
value that specifies the default width, in millimeters, for rendering a <a href="4be143af-2e99-41c5-894d-01902ed98673.htm">Report</a> record in a
physical-page oriented renderer. The default value is 0x00000000 millimeters.</p>
