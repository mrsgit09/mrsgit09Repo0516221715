<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.md"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.md"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapLineTemplate.OffsetY</b> element specifies the Y
offset in <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_0eca4039-3d5f-4fef-9ced-90ebd4d2bb22">map coordinates</a>
for map lines to which this <a href="37e2d016-be53-44eb-a5ae-5a01d6400909.md">MapLineTemplate</a>
is applied. This element is optional. If this element is present, its value
MUST be a <a href="c7d0946f-992e-4abc-a304-09b53e030692.md">Float</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.4) or an expression that evaluates to a <b>Float</b>. If this element is
not present, its value is interpreted as 0.</p>
