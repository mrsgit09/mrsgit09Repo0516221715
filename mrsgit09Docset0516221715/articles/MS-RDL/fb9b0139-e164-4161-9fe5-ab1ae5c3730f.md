<p>The <b>Report.Language</b> element specifies the primary
language of the text in a <a href="6bbaafec-020b-406c-b4e7-5e4318b616cb.htm">Report</a>.
This value is used as the default language for all language-dependent
expressions in the report. The <b>Report.Language</b> element is optional. If
this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. </p>
