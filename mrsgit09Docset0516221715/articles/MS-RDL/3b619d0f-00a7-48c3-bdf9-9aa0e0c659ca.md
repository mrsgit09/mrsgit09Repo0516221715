<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.htm"><b><i>RDL 2008/01</i></b></a><b><i>,

<p>The <b>ChartTitle.TextOrientation</b> element specifies the
orientation of the text of a <a href="67fc30a5-9c4a-4eaa-aec9-b2f734b240f5.htm">ChartTitle</a>.
This element is optional. If the <b>ChartTitle.TextOrientation</b> element is
present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.1) or an expression that evaluates to a <b>String</b>. The value of
this element MUST be one of the following or an expression that evaluates to
one of the following:</p>

<p><b>Auto</b>: Specifies whether the orientation is selected
automatically based on context (for example, Rotated270 for titles that are
docked on the left).</p>

<p><b>Horizontal</b>: Specifies horizontal text.</p>

<p><b>Rotated90</b>: Specifies vertical text, rotated 90
degrees.</p>

<p><b>Rotated270</b>: Specifies vertical text, rotated
270 degrees.</p>

<p><b>Stacked</b>: Specifies vertical text with no
character rotation.</p>

<p>If the <b>ChartTitle.TextOrientation</b> element is not
present, its value is interpreted as &quot;Auto&quot;.</p>
