<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.html"><b><i>RDL 2003/10</i></b></a><b>

<p>The <b>Details.Grouping</b> element specifies the
expressions by which to group the detail data in a <a href="660db744-699e-4ca3-a2d6-a5cab4bcf9b0.html">Table</a>. This element is
optional and is of type <a href="7d574154-eefe-4fc1-8b78-3a18b9350e87.html">Grouping</a>.
If the <b>Details.Grouping</b> element is not specified, the detail data is not
grouped.</p>
