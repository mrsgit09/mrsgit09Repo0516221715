<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.htm"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>TextRun.ActionInfo</b> element specifies the action
for a <a href="90623d67-443b-4480-9869-e03277a6223a.htm">TextRun</a>. This
element is optional. This element is ignored if an <a href="c2f76273-c025-42ba-bccf-91067d997228.htm">ActionInfo</a> is specified
for the parent <a href="469d0032-b5ec-43d9-ab36-d3a88b9cc1f6.htm">Textbox</a>,
even if the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_861707bc-950b-45dc-8ec3-a8afaf2c8545">text box</a>
action resolves to NULL. This element is of type <b>ActionInfo</b>.</p>
