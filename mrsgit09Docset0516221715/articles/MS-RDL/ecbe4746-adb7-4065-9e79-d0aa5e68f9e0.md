<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>MapPolygonTemplate.Style</b> element specifies style
information for map polygons (specified by <a href="3ee27e43-26a2-4f27-9a31-d97e374d8633.html">MapPolygon</a> elements) to
which a <a href="1b048418-d7ff-4c51-b08e-30ab8d5a63c5.html">MapPolygonTemplate</a>
is applied. This element is optional and is of type <a href="ea446209-9c6a-46ce-b472-fae8b8350b37.html">Style</a>.</p>
