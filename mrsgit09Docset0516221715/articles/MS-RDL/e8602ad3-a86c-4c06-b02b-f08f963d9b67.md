<p>The <b>MapColorScale.MapLocation</b> element specifies the
location of a <a href="fc14b477-a2d2-4048-843d-6a19beeb30bf.htm">MapColorScale</a>.
This element is optional. If this element is present, the <a href="8f016f89-8870-4463-83dc-0c56bc43972c.htm">MapColorScale.Position</a>
element is ignored. This element is of type <a href="5888ec40-7918-47d0-9b80-4d5897124957.htm">MapLocation</a>.</p>
