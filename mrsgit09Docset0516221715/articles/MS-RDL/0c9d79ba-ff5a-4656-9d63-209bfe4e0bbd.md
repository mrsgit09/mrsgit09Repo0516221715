<p>The <b>ChartSmartLabel.CalloutLineColor</b> element
specifies the color of a callout line. This element is optional. If this
element is present, its value MUST be an <a href="b302c6a5-6023-42b1-95ed-bafcdc4b5714.html">RdlColor</a> or an expression
that evaluates to an <b>RdlColor</b>. If this element is not present, its value
is interpreted as &quot;Black&quot;.</p>
