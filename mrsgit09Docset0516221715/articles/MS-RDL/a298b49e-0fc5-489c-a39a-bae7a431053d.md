<p>The <b>Rectangle.Top</b> element specifies the distance of a
<a href="e36a41ea-aeaf-45cc-969e-8ab1e380882c.html">Rectangle</a> from the top
of the rectangle's container. This element is optional. If this element is
present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.html">RdlSize</a>. </p>
