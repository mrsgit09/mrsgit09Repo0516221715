<p>The <b>Chart.DataElementName</b> element specifies the name
to use for the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_ea885285-50a5-4fab-ad19-90adda04d6c9">data element</a>
or attribute for a <a href="b0ab5524-7eb2-47a7-a4d3-230f5c8c5526.html">Chart</a>.
This element is optional. If the <b>Chart.DataElementName</b> element is
present, its value MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.</p>
