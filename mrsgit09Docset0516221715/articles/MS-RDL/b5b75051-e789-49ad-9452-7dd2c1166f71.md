will occur when an expression uses one of the aggregate functions with an
incorrect number of parameters. For example, this error will occur when a <a href="99982bda-2dd1-4626-b8ef-da888d95f4ff.htm">TextRun.Value</a> element
contains an expression &quot;=Sum()&quot;.</p>
