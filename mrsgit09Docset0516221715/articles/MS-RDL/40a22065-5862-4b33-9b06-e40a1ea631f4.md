<p>The <b>Line.RepeatWith</b> element specifies the name of a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_6abb146e-d02e-45aa-a034-b25b23b0dd48">data region</a> in which a line
is repeated if that data region spans multiple pages. The data region MUST be
in the same <a href="c5fef915-e842-43b4-91f9-56af4eb15be0.htm">ReportItems</a>
collection as this line. If the line is within a <a href="afff0921-7d95-4216-8f28-635c67d539d8.htm">PageSection</a> or <a href="ddc35223-1cb6-4136-823b-e72a3d12e1f9.htm">PageHeaderFooter</a>, this
element is ignored. The <b>Line.RepeatWith</b> element is optional. If this
element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1).</p>
