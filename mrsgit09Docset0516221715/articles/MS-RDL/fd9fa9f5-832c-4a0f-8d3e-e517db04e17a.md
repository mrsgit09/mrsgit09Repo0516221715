<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.md"><b><i>RDL 2003/10</i></b></a><b>

<p>The <b>ColumnGrouping.Height</b> element specifies the
height of a column header in a <a href="25419c0a-c7c6-43d7-8ca5-1af842666dcb.md">Matrix</a>. This element MUST
be specified. The value of this element MUST be a non-negative <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.md">RdlSize</a>.</p>
