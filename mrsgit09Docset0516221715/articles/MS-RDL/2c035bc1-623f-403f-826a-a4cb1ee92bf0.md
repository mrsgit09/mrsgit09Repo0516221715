<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>PinLabel.UseFontPercent</b> element specifies whether
the font size for a <a href="8a95fbbe-67d8-418f-8b2c-dc7fb18fdf6b.html">PinLabel</a>
instance is measured as a percentage of the font size of its ancestor <a href="86468d9f-c561-4b50-a689-5dfccfde8495.html">RadialScale</a> or <a href="744f8b40-7ad5-4652-94a1-76ae5df59389.html">LinearScale</a> element or, if
set to false, is measured in the units specified by the <a href="6e22f307-85f8-4212-b9d2-d73a613a1838.html">Style.FontSize</a> element of
the <a href="5a5d14e3-4d2e-418f-b955-3608c28056d8.html">PinLabel.Style</a>
element. </p>

<p>The <b>PinLabel.UseFontPercent</b> element is optional. If
this element is present, its value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.html">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2) or an expression that evaluates to a <b>Boolean</b>. If this element is
not present, its value is interpreted as false.</p>
