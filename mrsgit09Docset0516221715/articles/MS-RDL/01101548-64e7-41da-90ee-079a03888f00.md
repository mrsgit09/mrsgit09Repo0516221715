error that will occur when the value of the <b>Name</b> attribute of a <a href="fc2c2c96-ec36-47c2-b156-a6d8c0cbabd8.md">Variable</a> element within <a href="95ecdd74-a49c-402a-b34c-575008581c77.md">Report.Variables</a> is not
CLS-compliant <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.</p>
