<p>The <b>MapLine.MapFields</b> element specifies a collection
of <a href="fcd57c0c-4137-4771-89db-d56e3474ea36.htm">MapField</a> elements to
associate with a map line. The <b>MapLine.MapFields</b> element is optional.
This element is of type <a href="2c2c5097-27a6-4fd1-90a4-5c3545d00695.htm">MapFields</a>.</p>
