<p>The <b>MapPolygonLayer.MapPolygonRules</b> element specifies
a set of rules to be applied to the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_f553c04a-6cfa-4612-8395-c4f3af4a50ac">map polygons</a> (specified by <a href="3ee27e43-26a2-4f27-9a31-d97e374d8633.htm">MapPolygon</a> elements) in
the parent layer. This element is optional. This element is of type <a href="77b58882-2976-42cd-9e7a-aca2c6ee0139.htm">MapPolygonRules</a>.</p>
