expressions by which to sort the rows of data associated with a <a href="f01744d3-79fa-4f30-94bf-a1ffa6bde2ac.html">GaugePanel</a>. The <b>GaugePanel.SortExpressions</b>
element is optional. This element is of type <a href="6bc22842-81c9-45cb-bc37-58b09ed71578.html">SortExpressions</a>.</p>
