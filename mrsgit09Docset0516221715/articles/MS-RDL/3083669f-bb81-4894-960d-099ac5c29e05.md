<p>The <b>RadialScale.Name</b> attribute specifies the name for
a radial scale. This attribute MUST be specified. The value of this attribute
MUST be a case-sensitive <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>
within a particular <a href="250aef2b-c329-4faf-9bd9-2a66fdbb9731.html">RadialScales</a>
collection.</p>
