<p>The <b>ReportSection.Name</b> attribute specifies the name
to use for a <a href="96c3d25f-d8ce-4fe4-ab03-592edaa4a1da.htm">ReportSection</a>.
The <b>ReportSection.Name</b> attribute MUST be specified and MUST NOT be
specified more than once. The value of this attribute MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.</p>
