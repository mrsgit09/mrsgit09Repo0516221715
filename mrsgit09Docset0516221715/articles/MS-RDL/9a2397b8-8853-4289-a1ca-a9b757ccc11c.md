<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>MapColorPaletteRule.ShowInColorScale</b> element
specifies whether the colors that are assigned to map items are displayed in a <a href="fc14b477-a2d2-4048-843d-6a19beeb30bf.html">MapColorScale</a> instance.
The <b>MapColorPaletteRule.ShowInColorScale</b> element is optional. </p>

<p>If the <b>MapColorPaletteRule.ShowInColorScale</b> element
is present, its value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.html">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2) or an expression that evaluates to a <b>Boolean</b>. If this element is
not present, its value is interpreted as false. </p>
