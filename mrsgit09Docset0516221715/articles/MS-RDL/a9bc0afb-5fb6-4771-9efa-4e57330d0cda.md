<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.html"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.html"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>List.Name</b> attribute specifies a unique identifier
for a <a href="ea4c625c-0558-4fb3-b3b8-bde6c160b1e2.html">List</a>. This
attribute MUST be specified. The value of the <b>List.Name</b> attribute MUST
be a <a href="1b08f7bf-7112-4831-979a-fe3f8baae388.html">NormalizedString</a>
that is a case-sensitive <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.
This value MUST be unique among the set of all report item and <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_128cd76b-eecb-41e6-96f7-f73c2bf9235a">scope</a> names.</p>
