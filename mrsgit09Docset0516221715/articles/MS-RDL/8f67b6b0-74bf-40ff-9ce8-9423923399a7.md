<p>The <b>ChartEmptyPoints.AxisLabel</b> element specifies the
label to use on the axis of an empty data point. The <b>ChartEmptyPoints.AxisLabel</b>
element is optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. If this element is
not present, its value is interpreted as an empty string.</p>
