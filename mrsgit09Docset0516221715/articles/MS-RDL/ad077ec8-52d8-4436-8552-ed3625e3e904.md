<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapPointRules.MapMarkerRule</b> element specifies a
set of markers to be applied to map points in the parent layer based on the
value of the <a href="032181c1-561d-45e1-a933-772b6f3174bc.html">MapMarkerRule.DataValue</a>
element. The <b>MapPointRules.MapMarkerRule</b> element is optional. </p>

<p>If this element is present, it overrides <a href="9bb930a9-0ed2-4fc7-bf62-e5de125f46ff.html">MapMarkerTemplate.MapMarker</a>.
The <b>MapPointRules.MapMarkerRule</b> element is of type <a href="b7f81f81-be65-4bc2-8571-213ed55f2a92.html">MapMarkerRule</a>.</p>
