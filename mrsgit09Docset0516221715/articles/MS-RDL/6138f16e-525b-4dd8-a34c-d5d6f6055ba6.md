<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>Report.InitialPageName</b> element indicates the
initial value to use as the name of the paginated <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_3fc5c511-d6ab-4b29-a286-90bca2d65763">page</a> by the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_7b530e76-07fb-4f2d-a885-a60db51b2214">renderers</a>. This element is
optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>.</p>
