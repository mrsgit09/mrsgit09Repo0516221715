<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>PointerImage.MIMEType</b> element specifies the image
format of a <a href="ecf5073e-d4ae-4742-a92f-6790140b0fe6.html">PointerImage</a>.
The <b>PointerImage.MIMEType</b> element is optional. If this element is
present, its value MUST be a <a href="7e89fcbb-b433-48dd-819c-14d70e3b45bf.html">ReportMIMEType</a>.</p>

<p>If the peer element <a href="d95875ab-d00a-416b-ac72-c9fc81741720.html">PointerImage.Source</a> is set
to a value other than &quot;Database&quot;, the <b>PointerImage.MIMEType</b>
element is ignored.</p>
