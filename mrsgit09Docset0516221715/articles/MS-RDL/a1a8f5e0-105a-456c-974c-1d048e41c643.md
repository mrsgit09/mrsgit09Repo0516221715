start value of an <a href="b01d342e-1604-47c5-b90b-a4ce7bfd441c.md">IndicatorState</a>.
This element is ignored in any schema version prior to RDL 2010/01. This
element is of type <a href="9463d0dc-2309-420e-994e-47562e7670a1.md">GaugeInputValue</a>.</p>
