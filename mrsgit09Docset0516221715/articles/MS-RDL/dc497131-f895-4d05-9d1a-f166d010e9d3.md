<p>The <b>Report.Code</b> element specifies the code
definitions for custom functions to be used in an expression within a <a href="6bbaafec-020b-406c-b4e7-5e4318b616cb.html">Report</a>.<a id="Appendix_A_Target_3"></a><a href="1fe5fd87-2de5-4b2c-b762-5a4fd1373621.html#Appendix_A_3" aria-label="Product behavior note 3">&lt;3&gt;</a></p>

<p>The <b>Report.Code</b> element is optional. If this element
is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1). </p>
