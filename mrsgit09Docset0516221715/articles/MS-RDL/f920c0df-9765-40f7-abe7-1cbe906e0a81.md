<p>The <b>Rectangle.ToolTip</b> element specifies the tool tip
text for a <a href="e36a41ea-aeaf-45cc-969e-8ab1e380882c.htm">Rectangle</a>.
This element is optional. The value of this element MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>.</p>
