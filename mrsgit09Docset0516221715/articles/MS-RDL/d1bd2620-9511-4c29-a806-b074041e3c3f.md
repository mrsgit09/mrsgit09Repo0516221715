<p><b><i>Applies to </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.htm"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>CustomReportItem.Label</b> element specifies a
document map label to identify a <a href="6bb7b35c-e517-4444-a96b-9f2ccdd1a642.htm">CustomReportItem</a> within a
rendered report. This element is optional and MUST NOT be specified more than once.
If this element is specified, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>Variant</b>.</p>
