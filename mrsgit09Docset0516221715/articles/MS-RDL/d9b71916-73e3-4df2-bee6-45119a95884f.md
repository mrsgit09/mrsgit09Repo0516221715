<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>RadialScale.Width</b> element specifies the width of
a <a href="86468d9f-c561-4b50-a689-5dfccfde8495.html">RadialScale</a> as a
percentage of the <a href="2e113607-ee33-4abd-9ae3-6607c10d3c8a.html">RadialGauge</a>
size. This element is optional. If this element is present, its value MUST be a
<a href="c7d0946f-992e-4abc-a304-09b53e030692.html">Float</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.4) or an expression that evaluates to a <b>Float</b>. If this element is
not present, its value is interpreted as 0.</p>
