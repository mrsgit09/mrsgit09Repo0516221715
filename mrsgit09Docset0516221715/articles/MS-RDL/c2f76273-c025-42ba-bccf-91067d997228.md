        </div>
        <div id="mainSection">

<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>ActionInfo</b> element specifies a list of <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_b178b6c0-7df9-4107-95ca-12c7f0b9900b">actions</a> that are for a
report item.</p>

<p>In <a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.html">RDL 2003/10</a>
and <a href="3ebe2912-4958-4832-b391-cad1f5e13338.html">RDL 2005/01</a>,
the <a href="0c9b8d37-de61-420e-a652-26d3db8bc586.html">Action</a> element
specifies a list of actions for a report item.</p>
