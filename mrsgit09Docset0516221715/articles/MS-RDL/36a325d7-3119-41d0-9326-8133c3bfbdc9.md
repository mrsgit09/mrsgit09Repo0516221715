<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.htm"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.htm"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>List.Label</b> element specifies a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_46519c7a-96f3-40c7-9883-a17c87288632">document map</a> label to
identify a <a href="ea4c625c-0558-4fb3-b3b8-bde6c160b1e2.htm">List</a> within
the rendered <a href="6bbaafec-020b-406c-b4e7-5e4318b616cb.htm">Report</a>.
This element is optional.</p>

<p>If the <b>List.Label</b> element is present, its value MUST
be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_a3af3eaf-64b7-499b-a95f-193cd4c27812">Variant</a>. If the expression
returns NULL, no item is added to the document map. This element is ignored if
the list is contained within a page header or page footer.</p>
