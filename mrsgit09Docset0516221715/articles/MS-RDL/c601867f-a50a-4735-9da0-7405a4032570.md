<p><b><i>Applies to </i></b><a href="bf2bab1a-b608-4bcc-b718-1cc1baa9579c.htm"><b><i>RDL 2011/01</i></b></a></p>

<p>The <b>Image.Tag</b> element contains arbitrary additional
information about the image that clients can use to perform custom processing.
The <b>Image.Tag</b> element is optional and MUST NOT be specified more than
once. The value of the <b>Image.Tag</b> element MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) constant or an expression that evaluates to a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_a3af3eaf-64b7-499b-a95f-193cd4c27812">variant</a>.</p>
