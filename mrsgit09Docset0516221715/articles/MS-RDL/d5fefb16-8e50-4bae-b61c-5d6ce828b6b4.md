<p>The <b>GaugeInputValue.DataElementName</b> element specifies
the name to use for the data element or attribute for <a href="9463d0dc-2309-420e-994e-47562e7670a1.htm">GaugeInputValue</a> in a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_9069c206-b9e9-4374-a7ee-50faf5def25b">data rendering</a>. The <b>GaugeInputValue.DataElementName</b>
element is optional. This element MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>. The value of
this element is interpreted as a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1).</p>
