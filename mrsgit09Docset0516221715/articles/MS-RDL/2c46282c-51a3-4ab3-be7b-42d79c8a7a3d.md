<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>MapLineLayer.DataElementName</b> element specifies
the name for the <a href="37e2d016-be53-44eb-a5ae-5a01d6400909.htm">MapLineTemplate</a>
to use for a data element or attribute. This element is optional. If this
element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) that is a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.
If this element is not present, its value is interpreted as
&quot;MapDataRow&quot;.</p>
