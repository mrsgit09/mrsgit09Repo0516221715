identify a <a href="dbfff811-1be7-4e8b-a5d2-6cc522317fbe.html">Group</a> within
the rendered report. This element is optional and MUST NOT be specified more
than once. If this element is specified, it is of type <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a>.</p>
