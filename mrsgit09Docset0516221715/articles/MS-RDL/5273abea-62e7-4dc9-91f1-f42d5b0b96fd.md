<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>MapTitle.MapSize</b> element specifies the size of a <a href="9b8a7ec3-44b5-46d8-bdca-cb99308fa1f9.htm">MapTitle</a> element. The <b>MapTitle.MapSize</b>
element is optional. If this element is not present, the map title size MUST be
automatically calculated. This element is of type <a href="d34e1a7e-ada4-4989-9c0f-fbb69c9347ec.htm">MapSize</a>.</p>
