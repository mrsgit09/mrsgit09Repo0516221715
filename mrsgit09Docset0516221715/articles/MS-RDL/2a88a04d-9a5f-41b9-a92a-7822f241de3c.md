<p>The <b>MapPolygonLayer.MapMarkerTemplate</b> element specifies
a template for the <a href="3ee27e43-26a2-4f27-9a31-d97e374d8633.md">MapPolygon</a>
center point to be applied to points in the parent <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_bd6a41d0-83c8-46e0-8ecb-c7887094c205">map layer</a>. This element is
optional. This element is of type <a href="22055a42-2ec0-48cd-893f-f7bd717efc7a.md">MapMarkerTemplate</a>.</p>
