<p>The <b>TextRun.Label</b> element specifies a label for the
placeholder for a <a href="90623d67-443b-4480-9869-e03277a6223a.html">TextRun</a>.
If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1). The value of this element MUST appear as the display placeholder in the
designer tool UI.</p>
