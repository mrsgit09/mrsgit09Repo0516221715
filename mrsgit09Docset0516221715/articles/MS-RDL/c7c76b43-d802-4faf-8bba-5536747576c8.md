set of rules to be applied to the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_1e5099f9-65c2-4b23-934f-619318816172">map points</a> in the parent
layer. This element is optional. This element is of type <a href="d090d792-6d70-412c-b024-88c08de4d300.html">MapPointRules</a>.</p>
