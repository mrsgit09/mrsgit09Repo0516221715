<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>MapLineLayer.Name</b> attribute specifies a name for
the <a href="8681b1dc-d73e-4d35-b4fa-f7f459d4a304.htm">MapLineLayer</a>
element. The <b>MapLineLayer.Name</b> attribute MUST be specified. The value of
this attribute MUST be a case-sensitive <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) that is a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.</p>
