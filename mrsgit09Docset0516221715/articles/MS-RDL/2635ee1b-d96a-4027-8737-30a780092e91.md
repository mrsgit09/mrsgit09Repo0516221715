<p>The <b>MapLineTemplate.Style</b> element specifies style
information for map lines to which this <a href="37e2d016-be53-44eb-a5ae-5a01d6400909.html">MapLineTemplate</a> is
applied. This element is optional. This element is of type <a href="ea446209-9c6a-46ce-b472-fae8b8350b37.html">Style</a>.</p>
