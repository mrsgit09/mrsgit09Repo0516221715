<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>ChartAxis.MaxFontSize</b> element specifies the
maximum font size when auto-fitting axis labels. The <b>ChartAxis.MaxFontSize</b>
element is optional. If this element is present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.html">RdlSize</a> or an expression
that evaluates to an <b>RdlSize</b>. If this element is not present, its value
is interpreted as 10pt.</p>
