<p>The <b>TablixRow.Height</b> element specifies the height of
a row in a <a href="e42fb86e-799a-4202-8845-ac38831efccb.html">Tablix</a>. This
element MUST be specified exactly once and is of type <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.html">RdlSize</a>.</p>
