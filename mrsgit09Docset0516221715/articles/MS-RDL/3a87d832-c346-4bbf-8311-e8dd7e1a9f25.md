toggled by a <a href="469d0032-b5ec-43d9-ab36-d3a88b9cc1f6.htm">Textbox</a>.
This element is optional. This element is of type <a href="9505fbda-7f65-4874-a54a-1944059812e0.htm">Visibility</a>. </p>
