<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.htm"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.htm"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>Axis.Interlaced</b> element specifies whether strip
lines MUST be drawn at an interval of every other grid line for an <a href="2bfb943e-7cfe-41c1-baa4-5739a99a341b.htm">Axis</a>. If grid lines are
not used for the axis, the axis tick marks or labels are used to determine the
interval of the interlaced strip lines.</p>

<p>The <b>Axis.Interlaced</b> element is optional. If this
element is present, its value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.htm">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2). If this element is not present, its value is interpreted as false.</p>
