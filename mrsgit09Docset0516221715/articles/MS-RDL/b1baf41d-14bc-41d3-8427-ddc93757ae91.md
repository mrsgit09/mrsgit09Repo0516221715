<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>CustomLabel.Hidden</b> element specifies whether a <a href="519139e8-6188-4286-b148-dfd76a0a6be4.html">CustomLabel</a> is hidden. The
<b>CustomLabel.Hidden</b> element is optional. If this element is present, its
value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.html">Boolean</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.2) or an expression that evaluates to a <b>Boolean</b>. If this
element is not present, its value is interpreted as false.</p>
