<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>GaugeTickMarks.IntervalOffset</b> element defines the
offset for the first tick mark within a <a href="85bb1716-e94b-4d9a-97b3-3c681e0c53d5.html">GaugeTickMarks</a> element.
The <b>GaugeTickMarks.IntervalOffset</b> element is optional. If this element
is present, its value MUST be a <a href="c7d0946f-992e-4abc-a304-09b53e030692.html">Float</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.4) or an expression that evaluates to a <b>Float</b>. If this element is
not present, its <b>Float</b> value is interpreted as that of the <a href="4f58e97b-0bee-469f-b142-6ff68345a9f2.html">RadialScale.IntervalOffset</a>
or <a href="695cb1e7-ab91-4561-ba41-a8e145cf76b5.html">LinearScale.IntervalOffset</a>
value for the respective parent <a href="86468d9f-c561-4b50-a689-5dfccfde8495.html">RadialScale</a> or <a href="744f8b40-7ad5-4652-94a1-76ae5df59389.html">LinearScale</a> element. The <b>Float</b>
value of the <b>GaugeTickMarks.IntervalOffset</b> element MUST be non-negative.</p>
