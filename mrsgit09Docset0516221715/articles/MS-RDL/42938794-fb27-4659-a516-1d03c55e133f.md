<p>The <b>Textbox.CustomProperties</b> element specifies custom
information for a <a href="469d0032-b5ec-43d9-ab36-d3a88b9cc1f6.html">Textbox</a>
that will be handed to a report rendering component. This element is optional.
This element is of type <a href="93994776-7d8e-4cf2-932f-9c085f3deaf8.html">CustomProperties</a>.</p>
