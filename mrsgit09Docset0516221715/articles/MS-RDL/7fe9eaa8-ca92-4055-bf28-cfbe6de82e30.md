<p>The <b>ChartDataLabel.Rotation</b> element specifies the
angle of rotation for data label text. The <b>ChartDataLabel.Rotation</b>
element is optional. If this element is present, its value MUST be an <a href="176fbb59-c3e2-430c-b1bb-37fd15df813e.htm">Integer</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.3.17) or an expression that evaluates to an <b>Integer</b>. If this element
is not present, its value is interpreted as 0</p>
