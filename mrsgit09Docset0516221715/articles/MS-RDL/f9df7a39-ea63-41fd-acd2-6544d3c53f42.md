<p>The <b>Chart.DocumentMapLabel</b> element specifies a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_46519c7a-96f3-40c7-9883-a17c87288632">document map</a> label to
identify a <a href="b0ab5524-7eb2-47a7-a4d3-230f5c8c5526.html">Chart</a> within
the client UI to provide a user-friendly label for searching. The <b>Chart.DocumentMapLabel</b>
element is optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. </p>
