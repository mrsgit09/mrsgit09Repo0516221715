<p>The <b>Image.ToolTip</b> element specifies the tooltip text
for an <a href="63e1e5ab-7c49-4f62-8dbd-62d85de2b153.htm">Image</a>. This
element is optional. If this element is present, its value MUST be a String (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>.</p>
