<p>The <b>Rectangle.CustomProperties</b> element specifies
custom information for a <a href="e36a41ea-aeaf-45cc-969e-8ab1e380882c.html">Rectangle</a>
that will be handed to a report rendering component. The <b>Rectangle.CustomProperties</b>
element is optional. This element is of type <a href="93994776-7d8e-4cf2-932f-9c085f3deaf8.html">CustomProperties</a>.</p>
