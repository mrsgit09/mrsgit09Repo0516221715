<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>DataRowHierarchy</b> parent element defines the
hierarchy of <a href="f2f4d9bc-64dc-47dd-9515-c3f4e610af44.html">DataMember</a>
elements for row data of a <a href="6bb7b35c-e517-4444-a96b-9f2ccdd1a642.html">CustomReportItem</a>.
The definition and use of row data is determined by the custom report item.</p>
