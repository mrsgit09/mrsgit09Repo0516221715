<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.html"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.html"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>Matrix.CellDataElementName</b> element specifies the
name to use for the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_ea885285-50a5-4fab-ad19-90adda04d6c9">data
element</a> of each instance of the cell that contains the detail data of a <a href="25419c0a-c7c6-43d7-8ca5-1af842666dcb.html">Matrix</a> in a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_9069c206-b9e9-4374-a7ee-50faf5def25b">data rendering</a>. The <b>Matrix.CellDataElementName</b>
element is optional. If this element is present, its value MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>. If this
element is not present, its value is interpreted as &quot;Cell&quot;.</p>
