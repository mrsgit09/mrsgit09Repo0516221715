<p>The <b>Tablix.Left</b> element specifies the distance of the
<a href="e42fb86e-799a-4202-8845-ac38831efccb.html">Tablix</a> from the left of
the containing object. This element is optional and MUST NOT be specified more
than once. If this element is specified, it is of type <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.html">RdlSize</a>. If this element
is not specified, its value is interpreted as the minimum size.</p>
