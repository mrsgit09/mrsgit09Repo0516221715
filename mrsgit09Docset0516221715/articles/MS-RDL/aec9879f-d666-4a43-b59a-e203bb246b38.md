<p>The <b>MapColorRangeRule.DataElementName</b> element
specifies the name to use for a data element or attribute for the <a href="8812f9fc-af59-4901-97c5-243fb4032540.htm">MapColorRangeRule.DataValue</a>
element. The <b>MapColorRangeRule.DataElementName</b> element is optional. If
this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) that is a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.</p>
