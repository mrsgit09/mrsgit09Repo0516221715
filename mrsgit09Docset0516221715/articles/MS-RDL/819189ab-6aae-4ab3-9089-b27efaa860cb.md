<p>The <b>DataSetReference.DataSetName</b> element specifies
the name of the <a href="a14782b0-2e2f-4305-83a3-3de3fd750b6a.htm">DataSet</a>
that is being referenced. This element MUST be specified. The value of this
element MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.1).</p>
