<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.html"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.html"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>Grouping.Name</b> attribute specifies a unique
identifier for a <a href="7d574154-eefe-4fc1-8b78-3a18b9350e87.html">Grouping</a>.
This attribute MUST be specified. The value of the <b>Grouping.Name</b>
attribute MUST be a <a href="1b08f7bf-7112-4831-979a-fe3f8baae388.html">NormalizedString</a>
that is a case-sensitive <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.
This value MUST be unique among the set of all grouping, <a href="a14782b0-2e2f-4305-83a3-3de3fd750b6a.html">DataSet</a>, and data region
names.</p>
