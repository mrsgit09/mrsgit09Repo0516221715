<p>The <b>Map.CustomProperties</b> element specifies custom
information for a <a href="fd166dd8-6772-4507-b3f6-50a2b7cfd6ac.htm">Map</a>
that will be handed to a report rendering component. The <b>Map.CustomProperties</b>
element is optional. This element is of type <a href="93994776-7d8e-4cf2-932f-9c085f3deaf8.htm">CustomProperties</a>.</p>
