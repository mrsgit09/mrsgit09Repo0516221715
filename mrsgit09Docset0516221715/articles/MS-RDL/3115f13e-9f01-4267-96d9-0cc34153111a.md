<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.htm"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>ChartSmartLabel.CalloutLineWidth</b> element
specifies the width of a callout line. This element is optional.</p>

<p>If the <b>ChartSmartLabel.CalloutLineWidth</b> element is
present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.htm">RdlSize</a> or an expression
that evaluates to an <b>RdlSize</b>. If this element is not present, its value
is interpreted as 0.75pt.<a id="Appendix_A_Target_31"></a><a href="1fe5fd87-2de5-4b2c-b762-5a4fd1373621.htm#Appendix_A_31" aria-label="Product behavior note 31">&lt;31&gt;</a></p>
