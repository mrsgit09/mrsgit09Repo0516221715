<p>The <b>Rectangle.Name</b> attribute specifies a unique
identifier for a <a href="e36a41ea-aeaf-45cc-969e-8ab1e380882c.html">Rectangle</a>.
This attribute MUST be specified. The value of this attribute MUST be a
case-sensitive <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>
that is unique for all report items within a <a href="6bbaafec-020b-406c-b4e7-5e4318b616cb.html">Report</a>.</p>
