<p>The <b>Image.Top</b> element specifies the distance of an <a href="63e1e5ab-7c49-4f62-8dbd-62d85de2b153.htm">Image</a> from the top of the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_d6b55d1e-aea6-4b7e-a23d-c0de845e0b50">image</a> container, such as a <b>Rectangle</b>
or <b>Body</b>. This element is optional. If this element is present, its value
MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.htm">RdlSize</a>. </p>
