<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapPolygonLayer.VisibilityMode</b> element specifies
the visibility mode for a <a href="f54fa273-d9b2-4e49-a896-6001bcda016b.htm">MapPolygonLayer</a>.
This element is optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. </p>

<p>The value of this element MUST be one of the following or an
expression that evaluates to one of the following:</p>

<p><b>Visible</b>: The layer is always visible.</p>

<p><b>Hidden</b>: The layer is always hidden.</p>

<p><b>ZoomBased</b>: The layer is shown in a zoom level
range that is defined by <b>MapPolygonLayer.MaximumZoom</b> and <b>MapPolygonLayer.MinimumZoom</b>.
</p>

<p>If this element is not present, its value is interpreted as
&quot;Visible&quot;.</p>
