<p>The <b>Image.Bookmark</b> element specifies a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_42f9c2f4-8a4b-4d64-a0e1-fc071debdf4c">bookmark</a> for an <a href="63e1e5ab-7c49-4f62-8dbd-62d85de2b153.htm">Image</a> that can be linked
to via a bookmark <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_b178b6c0-7df9-4107-95ca-12c7f0b9900b">action</a>.
This element is optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>.</p>
