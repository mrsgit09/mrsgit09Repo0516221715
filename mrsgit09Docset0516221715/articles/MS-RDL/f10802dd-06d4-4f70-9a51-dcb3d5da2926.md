<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.htm"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>TopImage.MIMEType</b> element specifies the image
format of a <a href="d2a28b79-671f-4c62-bbca-fb90939f0846.htm">TopImage</a>
instance. The <b>TopImage.MIMEType</b> element is optional. If this element is
present, its value MUST be a <a href="7e89fcbb-b433-48dd-819c-14d70e3b45bf.htm">ReportMIMEType</a>.
If the peer element <a href="ba35d817-9cf7-4da9-a683-e1a1e36cb8ea.htm">TopImage.Source</a>
is set to a value other than &quot;Database&quot;, the <b>TopImage.MIMEType</b>
element is ignored.</p>
