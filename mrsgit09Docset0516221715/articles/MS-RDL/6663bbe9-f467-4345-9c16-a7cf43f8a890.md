<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapFieldNames.MapFieldName</b> element specifies a
field name to be imported from a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_acfac44c-899b-4e09-9b7b-02e9a82d4f50">shapefile</a> or from a
SQL Server dataset. This element MUST be specified. The value of this
element MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.1) or an expression that evaluates to a <b>String</b>.</p>
