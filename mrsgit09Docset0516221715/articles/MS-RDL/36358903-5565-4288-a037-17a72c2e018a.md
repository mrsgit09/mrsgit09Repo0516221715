<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>Subreport.OmitBorderOnPageBreak</b> element specifies
whether borders appear at locations where a <a href="04d4d6d6-e103-48fc-b4f7-bf5b4a7e56e5.html">Subreport</a> spans multiple
pages. This element also specifies whether <a href="b3c5d73d-2f29-4b32-9846-d077a22588bf.html">BackgroundImage</a> instances
within the subreport continue rather than restart after a page break. </p>

<p>This element is optional. If this element is present, its
value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.html">Boolean</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.2). If this element is not present, its value is interpreted as
false.</p>
