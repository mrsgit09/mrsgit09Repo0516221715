<p>The <b>Rectangle.Width</b> element specifies the width of a <a href="e36a41ea-aeaf-45cc-969e-8ab1e380882c.html">Rectangle</a>. This element is
optional. If this element is present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.html">RdlSize</a>. If this element
is not present, its value is interpreted as the width of the rectangle's
container (such as a <b>Rectangle</b> or <b>Body</b>) minus the value of the
peer <a href="b903fc31-a9f5-427d-89b8-9d6d61ea442b.html">Rectangle.Left</a>
element, if specified.</p>
