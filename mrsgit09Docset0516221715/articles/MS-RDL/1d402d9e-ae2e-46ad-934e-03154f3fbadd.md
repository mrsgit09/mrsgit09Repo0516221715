occur when an element that requires an <a href="6977536e-dae7-44f3-a737-a249567cf172.htm">RdlURL</a> value does not

&quot;https://&quot;, &quot;ftp://&quot;, &quot;mailto:&quot;, or
&quot;news:&quot;.</p>
