<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.md"><b><i>RDL 2008/01</i></b></a><b><i>,

<p>The <b>GaugePanel.Left</b> element specifies the distance of
a <a href="f01744d3-79fa-4f30-94bf-a1ffa6bde2ac.md">GaugePanel</a> from the
left of the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_8beb719e-adf3-461d-a3d4-d52ef83336ca">gauge panel</a>'s
container. The <b>GaugePanel.Left</b> element is optional. If this element is
present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.md">RdlSize</a>.</p>
