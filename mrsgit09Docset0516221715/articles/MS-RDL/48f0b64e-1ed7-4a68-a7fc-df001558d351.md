<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.htm"><b><i>RDL 2003/10</i></b></a><b>

<p>The <b>Chart.Label</b> element specifies a document map
label to identify an instance of a <a href="b0ab5524-7eb2-47a7-a4d3-230f5c8c5526.htm">Chart</a> within the client
UI. (This is done to provide a user-friendly label for searching.) This element
is optional.</p>

<p>If the <b>Chart.Label</b> element is present, its value MUST
be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. </p>
