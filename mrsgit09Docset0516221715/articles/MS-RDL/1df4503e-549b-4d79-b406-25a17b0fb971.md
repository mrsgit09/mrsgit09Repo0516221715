<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapColorScale.LabelInterval</b> element specifies the
interval between labels in a <a href="fc14b477-a2d2-4048-843d-6a19beeb30bf.html">MapColorScale</a>.
This element is optional. If this element is present, its value MUST be an <a href="176fbb59-c3e2-430c-b1bb-37fd15df813e.html">Integer</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.3.17) or an expression that evaluates to an <b>Integer</b>. If this element
is not present, its value is interpreted as 1.</p>
