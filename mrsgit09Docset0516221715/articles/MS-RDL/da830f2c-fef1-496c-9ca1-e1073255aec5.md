<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.md"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>MapPointLayer.MapDataRegionName</b> element specifies
the data region from which to consume data. This element is optional. This
element MUST be specified if the value of the <a href="e5028a82-02d1-4155-a9ab-78b7550dda05.md">MapPointLayer.MapSpatialDataRegion</a>
element is specified.</p>
