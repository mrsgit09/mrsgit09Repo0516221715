<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>PointerImage</b> element specifies a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_57f9ecc4-6264-41a2-bb02-b437f22d47e3">pointer image</a> to be used
for a <a href="19cdf02f-fcd5-41ca-b086-355eedb983b6.html">LinearPointer</a> or
a <a href="1446314e-813e-42f0-9a28-f1b96fd3a0da.html">RadialPointer</a> within
a <a href="021b569b-07ae-462a-ac62-d3ab51f183f5.html">LinearGauge</a> or <a href="2e113607-ee33-4abd-9ae3-6607c10d3c8a.html">RadialGauge</a> instance. The <b>PointerImage</b>
element is optional.</p>
