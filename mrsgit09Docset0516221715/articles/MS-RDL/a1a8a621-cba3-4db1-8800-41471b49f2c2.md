<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>LinearPointers</b> element specifies the collection
of <a href="19cdf02f-fcd5-41ca-b086-355eedb983b6.html">LinearPointer</a>
instances for a <a href="744f8b40-7ad5-4652-94a1-76ae5df59389.html">LinearScale</a>.
The <b>LinearPointers</b> element is optional. If this element is present, it
MUST contain at least one <a href="f3a5296b-a710-401b-9239-b7e16f96badb.html">LinearPointers.LinearPointer</a>
instance.</p>
