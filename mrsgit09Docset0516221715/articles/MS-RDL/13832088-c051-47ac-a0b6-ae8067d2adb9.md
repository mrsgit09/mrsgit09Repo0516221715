<p>The <b>CapImage.TransparentColor</b> element specifies the
color to be treated as transparent in a <a href="aeadc8ff-7b09-41e1-9ab8-2a3343bf25bb.html">CapImage</a>. This element is
optional. If present, the value of the <b>CapImage.TransparentColor</b> element
MUST be an <a href="b302c6a5-6023-42b1-95ed-bafcdc4b5714.html">RdlColor</a> or
an expression that evaluates to an <b>RdlColor</b>.</p>
