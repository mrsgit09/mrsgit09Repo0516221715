<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p><i>rsNestedAggregateViaLookup</i> is a critical error that
will occur when an aggregate function references a <a href="f7cfa0a3-695f-496c-ac72-e4f865e2803a.html">Lookup</a>, <a href="def44c38-e9cc-449b-87fc-72a95ef1c8fb.html">LookupSet</a>, or <a href="5b2699f6-8b46-40d7-9a92-0d23132d3d08.html">MultiLookup</a> function in
its <i>Expression</i> parameter and the <b>Lookup</b>, <b>LookupSet</b>, or <b>MultiLookup</b>
function references an aggregate function in its <i>Source</i> expression
parameter.</p>
