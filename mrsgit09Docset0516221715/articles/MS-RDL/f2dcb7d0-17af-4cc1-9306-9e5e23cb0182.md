<p>The <b>Subreport.DataElementName</b> element specifies the
name to use for the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_ea885285-50a5-4fab-ad19-90adda04d6c9">data
element</a> or attribute of a <a href="04d4d6d6-e103-48fc-b4f7-bf5b4a7e56e5.md">Subreport</a>. This element is
optional. If this element is present, it MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>. If this
element is not present, its value defaults to the name of the subreport.</p>
