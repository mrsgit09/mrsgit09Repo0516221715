<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.md"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.md"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.md"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>LinearScale.Width</b> element specifies the width for
a <a href="744f8b40-7ad5-4652-94a1-76ae5df59389.md">LinearScale</a> as a
percentage of <a href="021b569b-07ae-462a-ac62-d3ab51f183f5.md">LinearGauge</a>
size. The <b>LinearScale.Width</b> element is optional. If this element is
present, its value MUST be a <a href="c7d0946f-992e-4abc-a304-09b53e030692.md">Float</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.4) or an expression that evaluates to a <b>Float</b>. If this
element is not present, its value is interpreted as 0.</p>
