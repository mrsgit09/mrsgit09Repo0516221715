<p>The <b>GaugeInputValue.Value</b> element specifies an
expression for the value of a <a href="9463d0dc-2309-420e-994e-47562e7670a1.md">GaugeInputValue</a> instance.
The <b>GaugeInputValue.Value</b> element MUST be specified. The value of the <b>GaugeInputValue.Value</b>
element MUST be a <b>Numeric</b> or an expression that evaluates to a <b>Numeric</b>.</p>
