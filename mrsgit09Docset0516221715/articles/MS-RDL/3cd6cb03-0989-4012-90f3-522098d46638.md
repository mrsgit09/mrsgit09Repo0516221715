<p>The <b>Parameter.Name</b> attribute specifies the name of
the <a href="7c3f4c83-9172-48db-94c1-693295c5d623.html">ReportParameter</a> of
a dependent <a href="6bbaafec-020b-406c-b4e7-5e4318b616cb.html">Report</a>.
This attribute MUST be specified. The value of this attribute MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.</p>
