<p>The <b>Group.Filters</b> element specifies a collection of <a href="c0f6a66a-1055-4f4d-b1e7-4fc47b588ed2.htm">Filter</a> elements for a <a href="dbfff811-1be7-4e8b-a5d2-6cc522317fbe.htm">Group</a>. The <b>Group.Filters</b>
element is optional and MUST NOT be specified more than once. If the <b>Group.Filters</b>
element is specified, it is of type <a href="4075354a-2747-4ce0-ba0f-3e32a950f605.htm">Filters</a>.</p>
