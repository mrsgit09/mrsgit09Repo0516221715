<p>The <b>Actions.Action</b> element specifies an action that
is associated with a report item. This element MUST be specified at least once
within an <a href="df7671f8-303b-497f-8cb7-c6aaf1b2ac78.htm">Actions</a>
collection. This element is of type <a href="0c9b8d37-de61-420e-a652-26d3db8bc586.htm">Action</a>.</p>
