<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.htm"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>PageSection.PrintBetweenSections</b> element
specifies whether the <a href="afff0921-7d95-4216-8f28-635c67d539d8.htm">PageSection</a>
should be shown on the rendered <a href="b5e525d5-00d6-4e1a-8813-55f327da6b4c.htm">Page</a> in a report in
RDL 2008/01 or a <a href="96c3d25f-d8ce-4fe4-ab03-592edaa4a1da.htm">ReportSection</a>
in RDL 2010/01 and RDL 2016/01. The <b>PageSection.PrintBetweenSections</b>
element is optional. If this element is present, its value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.htm">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2). If this element is not present, its value is interpreted as false.</p>
