<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapLineLayer.VisibilityMode</b> element specifies the
visibility mode for a map <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_d18f341f-9a11-41e7-bc17-fa40808259cc">line
layer</a>. This element is optional. If this element is present, its value MUST
be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. </p>

<p>The value of this element MUST be one of the following or an
expression that evaluates to one of the following:</p>

<p><b>Visible</b>: The layer is always visible.</p>

<p><b>Hidden</b>: The layer is always hidden.</p>

<p><b>ZoomBased</b>: The layer is shown in a zoom level
range defined by <a href="9b25ed11-6b85-445f-b674-aca492ce1253.htm">MapLineLayer.MaximumZoom</a>
and <a href="24bd3dbc-3166-4234-b418-dcc9f0468af7.htm">MapLineLayer.MinimumZoom</a>.
</p>

<p>If this element is not present, its value is interpreted as
&quot;Visible&quot;.</p>
