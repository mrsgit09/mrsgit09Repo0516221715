<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>RadialScale.Interval</b> element specifies the
default interval between <a href="85bb1716-e94b-4d9a-97b3-3c681e0c53d5.html">GaugeTickMarks</a>
instances and <a href="a827496f-21f1-4ecd-925c-8ef98c6dbb10.html">RadialScale.ScaleLabels</a>
instances within a <a href="86468d9f-c561-4b50-a689-5dfccfde8495.html">RadialScale</a>.
The <b>RadialScale.Interval</b> element is optional. If this element is
present, its value MUST be a <a href="c7d0946f-992e-4abc-a304-09b53e030692.html">Float</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.4) or an expression that evaluates to a <b>Float</b>. The value of
this element MUST be greater than or equal to 0 and less than or equal
to 1.79769<sup>+308</sup>.</p>

<p>If this element is not present, its value is interpreted
as 0.</p>
