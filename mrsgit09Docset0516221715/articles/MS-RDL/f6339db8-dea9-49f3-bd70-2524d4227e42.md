<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

<p>The <b>GaugePanel.DocumentMapLabel</b> element specifies a
label to identify a <a href="f01744d3-79fa-4f30-94bf-a1ffa6bde2ac.html">GaugePanel</a>
within the client UI in order to provide a user-friendly label for searching.
The <b>GaugePanel.DocumentMapLabel</b> element is optional. If this element is
present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.1) or an expression that evaluates to a <b>String</b>. This element
MUST NOT be contained within a <a href="afff0921-7d95-4216-8f28-635c67d539d8.html">PageSection</a>.</p>
