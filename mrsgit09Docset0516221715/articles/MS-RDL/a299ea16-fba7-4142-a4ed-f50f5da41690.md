<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.htm"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>CustomLabel.RotateLabel</b> element specifies whether
the text for a <a href="519139e8-6188-4286-b148-dfd76a0a6be4.htm">CustomLabel</a>
rotates along with its containing element. The <b>CustomLabel.RotateLabel</b>
element is optional. </p>

<p>If this element is present, its value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.htm">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2) or an expression that evaluates to a <b>Boolean</b>. If this element is
not present, its value is interpreted as false. If the containing element is
not a <a href="86468d9f-c561-4b50-a689-5dfccfde8495.htm">RadialScale</a>, the <b>CustomLabel.RotateLabel</b>
element is ignored.</p>
