<p>The <b>Textbox.ActionInfo</b> element specifies one or more <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_b178b6c0-7df9-4107-95ca-12c7f0b9900b">actions</a> (such as a
hyperlink) associated with a <a href="469d0032-b5ec-43d9-ab36-d3a88b9cc1f6.md">Textbox</a>.
This element is optional. This element is of type <a href="c2f76273-c025-42ba-bccf-91067d997228.md">ActionInfo</a>.</p>
