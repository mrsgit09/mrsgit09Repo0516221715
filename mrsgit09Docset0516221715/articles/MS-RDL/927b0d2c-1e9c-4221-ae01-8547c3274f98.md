<p>The <b>LinearPointer.Type</b> element specifies the type of
a <a href="19cdf02f-fcd5-41ca-b086-355eedb983b6.html">LinearPointer</a>. The <b>LinearPointer.Type</b>
element is optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. The value of this
element MUST be one of the following or an expression that evaluates to one of
the following:</p>

<p><b>Marker</b>: Specifies that the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_9fc838d7-70b1-4d0e-a8d7-20ffff39919f">linear pointer</a> is a marker.</p>

<p><b>Bar</b>: Specifies that the linear pointer is a
bar.</p>

<p><b>Thermometer</b>: Specifies that the linear pointer
is a thermometer.</p>

<p>If the <b>LinearPointer.Type</b> element is not present, its
value is interpreted as &quot;Marker&quot;.</p>
