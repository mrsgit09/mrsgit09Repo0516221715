<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.html"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.html"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>Table.Left</b> element specifies the distance of a <a href="660db744-699e-4ca3-a2d6-a5cab4bcf9b0.html">Table</a> from the left of the
table's container. This element is optional. If this element is specified, its
value MUST be a non-negative <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.html">RdlSize</a>.
If the <b>Table.Left</b> element is not specified, the value of this element is
interpreted as 0.</p>
