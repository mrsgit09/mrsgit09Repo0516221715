<p>The <b>Textbox.DocumentMapLabel</b> element specifies a
label to identify a <a href="469d0032-b5ec-43d9-ab36-d3a88b9cc1f6.htm">Textbox</a>
within the client UI in order to provide a user-friendly label for searching.
This element is optional. If the <b>Textbox.DocumentMapLabel</b> element is
present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a>
(<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a>
section 3.2.1) or an expression that evaluates to a <b>String</b>. This element
is ignored when used within a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.htm#gt_861707bc-950b-45dc-8ec3-a8afaf2c8545">text
box</a> that is contained within a <a href="afff0921-7d95-4216-8f28-635c67d539d8.htm">PageSection</a> or <a href="ddc35223-1cb6-4136-823b-e72a3d12e1f9.htm">PageHeaderFooter</a>.</p>
