<p>The <b>Report.DataElementName</b> element specifies the name
to use for the top-most element in a report. This element is optional. If this
element is present, its value MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>. </p>
