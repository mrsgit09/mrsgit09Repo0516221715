<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.htm"><b><i>RDL 2003/10</i></b></a><b>

<p>The <b>Table.PageBreakAtEnd</b> element specifies that a
renderer inserts a page break at the end of this <a href="660db744-699e-4ca3-a2d6-a5cab4bcf9b0.htm">Table</a>. This element is
optional. The value of the <b>Table.PageBreakAtEnd</b> element MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.htm">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2). If this element is not specified, its value is interpreted as false.</p>
