<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.md"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.md"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>StateIndicator.StateDataElementName</b> element
specifies the name to use for the data element or attribute in a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_9069c206-b9e9-4374-a7ee-50faf5def25b">data rendering</a> for the
indicator state name that the state indicator value falls in its range. The <b>StateIndicator.StateDataElementName</b>
element is optional. This element MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>. The value of
this element is interpreted as a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.md">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1).</p>
