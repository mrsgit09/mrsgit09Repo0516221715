<p><b><i>Applies to </i></b><a href="bf2bab1a-b608-4bcc-b718-1cc1baa9579c.html"><b><i>RDL 2011/01</i></b></a></p>

<p>The <b>Group.Relationship</b> element specifies a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_2913b24a-aa1a-42cb-8b80-047821e296cb">relationship</a> to use for
correlating data in this <a href="dbfff811-1be7-4e8b-a5d2-6cc522317fbe.html">Group</a>
with the data in the containing scope. The <b>Group.Relationship</b> element is
optional and MUST NOT be specified more than once. If this element is
specified, it is of type <a href="6d1c77e5-1573-4ad6-8d2a-c507411ad94b.html">Relationship</a>.
This element is ignored if the <a href="a14782b0-2e2f-4305-83a3-3de3fd750b6a.html">DataSet</a> for this <b>Group</b>
is the same as the <b>DataSet</b> for each containing scope.</p>
