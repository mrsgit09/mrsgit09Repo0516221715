<p>The <b>ChartLegend.ChartElementPosition</b> element
specifies a custom position for a <a href="68a0757c-8f1a-42b9-9473-ccedd40029fb.htm">ChartLegend</a>. The <b>ChartLegend.ChartElementPosition</b>
element is optional. If this element is not present, automatic positioning is
used. This element is of type <a href="ea978f67-64fd-48c2-af63-ef1752bfedec.htm">ChartElementPosition</a>.</p>
