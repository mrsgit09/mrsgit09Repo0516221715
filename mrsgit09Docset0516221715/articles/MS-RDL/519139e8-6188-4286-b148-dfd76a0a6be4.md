and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>CustomLabel</b> element specifies a custom label for
a <a href="86468d9f-c561-4b50-a689-5dfccfde8495.htm">RadialScale</a> or a <a href="744f8b40-7ad5-4652-94a1-76ae5df59389.htm">LinearScale</a> element. The <b>CustomLabel</b>
element MUST be specified at least once within a <a href="b563b913-9d20-4bc8-b366-4558c8ca280f.htm">CustomLabels</a> collection.</p>
