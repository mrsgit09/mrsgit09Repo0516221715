<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapTitle.MapLocation</b> element specifies the
location of a <a href="9b8a7ec3-44b5-46d8-bdca-cb99308fa1f9.html">MapTitle</a>
element. The <b>MapTitle.MapLocation</b> element is optional and is of type <a href="5888ec40-7918-47d0-9b80-4d5897124957.html">MapLocation</a>. If the <b>MapTitle.MapLocation</b>
element is present, the <a href="024ed767-0d6b-4b99-a641-9600624ef5e4.html">MapTitle.Position</a>
element is ignored. </p>
