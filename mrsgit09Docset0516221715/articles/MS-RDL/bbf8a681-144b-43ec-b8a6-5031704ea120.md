<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

<p>The <b>Page.Style</b> element specifies style information
for a <a href="b5e525d5-00d6-4e1a-8813-55f327da6b4c.html">Page</a> in a report
in RDL 2008/01, or for a <a href="96c3d25f-d8ce-4fe4-ab03-592edaa4a1da.html">ReportSection</a> in
RDL 2010/01 and RDL 2016/01. The <b>Page.Style</b> element is
optional. This element is of type <a href="ea446209-9c6a-46ce-b472-fae8b8350b37.html">Style</a>.</p>
