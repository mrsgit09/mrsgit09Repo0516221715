<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>ChartLegend.TextWrapThreshold</b> element specifies
the number of characters after which to wrap text in a <a href="68a0757c-8f1a-42b9-9473-ccedd40029fb.html">ChartLegend</a>.</p>

<p>The <b>ChartLegend.TextWrapThreshold</b> element is
optional. If this element is present, its value MUST be a non-negative <a href="176fbb59-c3e2-430c-b1bb-37fd15df813e.html">Integer</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.3.17) or an expression that evaluates to a non-negative <b>Integer</b>. If
this element is not present, its value is interpreted as 25.</p>
