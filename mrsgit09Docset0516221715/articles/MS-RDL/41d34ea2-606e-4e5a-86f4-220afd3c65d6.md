<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapLineTemplate.DataElementLabel</b> element
specifies the label for the <a href="37e2d016-be53-44eb-a5ae-5a01d6400909.html">MapLineTemplate</a>
to use for a data element or attribute. This element is optional. If this
element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. If this element is
not present, its value is interpreted as the value of the <a href="79987dab-fc00-40ce-bfa7-3f727478b241.html">MapLineTemplate.Label</a>
property. </p>
