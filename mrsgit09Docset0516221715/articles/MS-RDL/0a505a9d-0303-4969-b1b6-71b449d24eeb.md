<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>ReportSection.DataElementName</b> element specifies
the name to use for the element that represents this <a href="96c3d25f-d8ce-4fe4-ab03-592edaa4a1da.html">ReportSection</a> in a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_9069c206-b9e9-4374-a7ee-50faf5def25b">data rendering</a>. The <b>ReportSection.DataElementName</b>
element is optional. If this element is present, its value MUST be a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>.</p>
