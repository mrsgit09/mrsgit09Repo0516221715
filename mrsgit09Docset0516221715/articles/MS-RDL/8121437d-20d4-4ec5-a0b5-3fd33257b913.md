<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.htm"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.htm"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>Matrix.PageBreakAtEnd</b> element specifies whether a
renderer inserts a page break at the end of this <a href="25419c0a-c7c6-43d7-8ca5-1af842666dcb.htm">Matrix</a>. This element is
optional. The value of <b>Matrix.PageBreakAtEnd</b> MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.htm">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2). If this element is not present, its value is interpreted as false.</p>
