<p>The <b>Rectangle.Bookmark</b> element specifies a bookmark
for a <a href="e36a41ea-aeaf-45cc-969e-8ab1e380882c.html">Rectangle</a> that
can be linked to via an <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_b178b6c0-7df9-4107-95ca-12c7f0b9900b">action</a>
with the <a href="37a720d6-0c3b-4a13-b909-74c59ef3ebed.html">Action.BookmarkLink</a>
element set. This element is optional. If this element is present, its value
MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>.</p>
