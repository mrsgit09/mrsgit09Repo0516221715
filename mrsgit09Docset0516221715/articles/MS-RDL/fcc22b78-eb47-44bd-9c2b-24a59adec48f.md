<p>The <b>ChartSeries.ChartMarker</b> element specifies the
appearance of a data point marker. This element is optional. If the parent of
the <a href="aee11573-3fcf-4365-938b-e6c8ceece6e1.html"><span>ChartSeries</span></a> is not <a href="1d639ad1-8e24-45ec-8dcb-8b6163780a36.html">ChartDerivedSeries</a>, the <b>ChartSeries.ChartMarker</b>
element is ignored.</p>

<p>This element is of type <a href="82987908-050f-4a6d-a8be-d6cc28a34d62.html">ChartMarker</a>.</p>
