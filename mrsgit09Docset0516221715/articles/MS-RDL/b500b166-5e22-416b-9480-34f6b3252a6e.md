<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.html"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>TablixMember.KeepTogether</b> element specifies
whether an attempt is made to render instances of a <a href="1d8a9691-b173-4e24-9ea9-1f486bc824fd.html">TablixMember</a> on a single
page. This element is optional and MUST NOT be specified more than once.</p>

<p>If the <b>TablixMember.KeepTogether</b> element is
specified, its value MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.html">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2). If the value of this element is true, instances of the <b>TablixMember</b>
SHOULD<a id="Appendix_A_Target_12"></a><a href="1fe5fd87-2de5-4b2c-b762-5a4fd1373621.html#Appendix_A_12" aria-label="Product behavior note 12">&lt;12&gt;</a> be rendered on a single page.
If the <b>TablixMember.KeepTogether</b> element is not specified, its value is
interpreted as false.</p>
