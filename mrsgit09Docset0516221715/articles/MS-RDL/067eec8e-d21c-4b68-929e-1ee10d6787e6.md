<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.htm"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.htm"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapLegend.MinFontSize</b> element specifies the
minimum font size for the auto-sizing algorithm of <a href="71c7ce11-4e8a-433b-975a-731e089ea04f.htm">MapLegend</a> text. This
element is optional. </p>

<p>If this element is present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.htm">RdlSize</a> or an expression
that evaluates to an <b>RdlSize</b>. If this element is not present, its value
is interpreted as 7pt. This element is ignored if the value of the <a href="ee6cf3af-8c09-4344-bc1b-455ce1954cd7.htm">MapLegend.AutoFitTextDisabled</a>
element is true.</p>
