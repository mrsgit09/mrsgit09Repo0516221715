<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.htm"><b><i>RDL 2003/10</i></b></a><b>

<p>The <b>List.ToolTip</b> element specifies a tooltip for a <a href="ea4c625c-0558-4fb3-b3b8-bde6c160b1e2.htm">List</a>. This element is
optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.htm">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. If this element is
not specified, its value is interpreted as NULL.</p>
