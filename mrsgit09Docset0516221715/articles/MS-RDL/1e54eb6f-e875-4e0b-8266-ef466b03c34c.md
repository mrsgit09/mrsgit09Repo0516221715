namespace to use for a <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_9069c206-b9e9-4374-a7ee-50faf5def25b">data
rendering</a>. The <b>Report.DataSchema</b> element is optional. If this
element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1).</p>
