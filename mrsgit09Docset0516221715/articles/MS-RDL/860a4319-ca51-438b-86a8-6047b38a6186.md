<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.htm"><b><i>RDL 2008/01</i></b></a><b><i>,

<p>The <b>RadialGauge.PivotY</b> element specifies the
y-position of the pivot point as a percentage of <a href="1cacf95d-9dcd-4fb0-b23a-56f15a7f0e4b.htm">RadialGauge.Height</a> value
for a <a href="2e113607-ee33-4abd-9ae3-6607c10d3c8a.htm">RadialGauge</a>. The <b>RadialGauge.PivotY</b>
element is optional. If this element is present, its value MUST be a <a href="c7d0946f-992e-4abc-a304-09b53e030692.htm">Float</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.4) or an expression that evaluates to a <b>Float</b>. If this element is
not present, its value is interpreted as 50.</p>
