<p>The <b>Subreport.Left</b> element specifies the distance of
the subreport from the left of the subreport’s container, such as a <a href="e36a41ea-aeaf-45cc-969e-8ab1e380882c.html">Rectangle</a> or <a href="6bf4e125-fdfd-4d04-88aa-c4395ba8a252.html">Body</a>. This element is
optional. If this element is present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.html">RdlSize</a>. </p>
