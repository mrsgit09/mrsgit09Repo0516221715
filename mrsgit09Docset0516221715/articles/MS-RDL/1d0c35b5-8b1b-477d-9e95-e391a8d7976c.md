<p>The <b>CustomReportItem.Name</b> attribute specifies a
unique identifier for a <a href="6bb7b35c-e517-4444-a96b-9f2ccdd1a642.html">CustomReportItem</a>.
The <b>CustomReportItem.Name</b> attribute MUST be specified, and its value
MUST be a <a href="1b08f7bf-7112-4831-979a-fe3f8baae388.html">NormalizedString</a>
that is a case-sensitive <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.html#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant
identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a>
that is unique among all report item names and scope names.</p>
