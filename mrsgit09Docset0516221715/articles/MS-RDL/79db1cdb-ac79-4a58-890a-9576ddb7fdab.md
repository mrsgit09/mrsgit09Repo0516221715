<p>The <b>PointerImage.HueColor</b> element specifies the color
to tint a <a href="ecf5073e-d4ae-4742-a92f-6790140b0fe6.html">PointerImage</a>.
This element is optional. If this element is present, its value MUST be an <a href="b302c6a5-6023-42b1-95ed-bafcdc4b5714.html">RdlColor</a> or an expression
that evaluates to an <b>RdlColor</b>.</p>
