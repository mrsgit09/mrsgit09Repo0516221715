<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.html"><b><i>RDL 2010/01</i></b></a><b><i>
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.html"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>MapColorScale.LabelBehavior</b> element specifies how
the labels display the values in a <a href="fc14b477-a2d2-4048-843d-6a19beeb30bf.html">MapColorScale</a>. This
element is optional. If this element is present, its value MUST be a <a href="1ed81ef3-a683-45e3-aaad-bd2bbe71bc3d.html">String</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.1) or an expression that evaluates to a <b>String</b>. </p>

<p>The value of this element MUST be one of the following or an
expression that evaluates to one of the following:</p>

<p><b>Auto:</b> The value is automatically determined.</p>

<p><b>ShowMiddleValue:</b> The middle value is shown.</p>

<p><b>ShowBorderValue:</b> The border value is shown.</p>

<p>If this element is not present, its value is interpreted as
&quot;Auto&quot;.</p>
