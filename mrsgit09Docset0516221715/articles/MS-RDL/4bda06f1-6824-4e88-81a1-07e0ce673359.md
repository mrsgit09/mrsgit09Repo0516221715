<p>The <b>Paragraph.SpaceBefore</b> element specifies the
vertical spacing before a <a href="c813d832-e92f-40e9-aadf-77ec1845efbb.htm">Paragraph</a>.
This element is optional. If this element is present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.htm">RdlSize</a> or an expression
that evaluates to an <b>RdlSize</b>.</p>
