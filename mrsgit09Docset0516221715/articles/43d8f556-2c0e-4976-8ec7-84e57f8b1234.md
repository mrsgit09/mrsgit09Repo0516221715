<p><b>dwPadding (ANSI file format only, 4 bytes):</b>
Unused padding; MUST be set to zero.</p>

<p><b>rgbAMapBits (496 bytes):</b> <b>AMap</b> data.
This is represented as a sequence of bits that marks whether blocks of 64 bytes
of data have been allocated. If the n<sup>th</sup> bit is set to 1, then the n<sup>th</sup>
block of 64 bytes has been allocated. Alternatively, if the n<sup>th</sup> bit
is set to 0, the n<sup>th</sup> block of 64 bytes is not allocated (free).</p>

<p><b>pageTrailer (Unicode: 16 bytes; ANSI: 12 bytes):</b>
A <b>PAGETRAILER</b> structure (section <a href="f4ccb38a-930a-4db4-98df-a69c195926ba.htm">2.2.2.7.1</a>). The <b>ptype</b>
subfield of <b>pageTrailer</b> MUST be set to <b>ptypeAMap</b>. The other
subfields of <b>pageTrailer</b> MUST be set as specified in section 2.2.2.7.1.</p>
