<p>The Report Server Web Service for Report Management for
SharePoint Mode: ReportService2006 protocol is an API protocol that is provided
by the ReportService2006 web service for managing objects and settings on a <a href="755aec02-e59f-4377-9100-4673bbf7b123.htm#gt_cbdd3a12-e9ec-43e2-ac97-9c47f171f96a">report server</a> that is
configured for SharePoint integrated mode.</p>

normative. All other sections and examples in this specification are
informative.</p>
