<p>The Outlook Personal Folders (.pst) File Format specifies
the necessary technical information required to read and write the contents of
a Personal Folders File (PST). This document also specifies the minimum
requirements for a PST file to be recognizable as valid in order for
implementers to create PST files that can be mounted and used by other
implementations of the specification.</p>

<p>Sections 1.7 and 2 of this specification are normative. All
other sections and examples in this specification are informative.</p>
