<p><b>btype (1 byte):</b> Block type; MUST be set to
0x02.</p>

<p><b>cLevel (1 byte):</b> MUST be set to 0x00.</p>

<p><b>cEnt (2 bytes):</b> The number of SLENTRYs in the
SLBLOCK. This value and the number of elements in the <b>rgentries</b> array
MUST be non-zero. When this value transitions to zero, it is required for the
block to be deleted.</p>

<p><b>dwPadding (4 bytes, Unicode only):</b> Padding;
MUST be set to zero.</p>

<p><b>rgentries (variable size):</b> Array of <b>SLENTRY</b>
structures. The size is equal to the number of entries indicated by <b>cEnt</b>
multiplied by the size of an <b>SLENTRY</b> (24 bytes for Unicode PST files, 12
bytes for ANSI PST Files).</p>

<p><b>rgbPadding (optional, variable):</b> This field is
present if the total size of all of the other<b> </b>fields is not a multiple
of 64. The size of this field is the smallest number of bytes required to make
the size of the SLBLOCK a multiple of 64. Implementations MUST ignore this
field.</p>

<p><b>blockTrailer (ANSI: 12 bytes; Unicode: 16 bytes):</b>
A <b>BLOCKTRAILER</b> structure (section <a href="a14943ef-70c2-403f-898c-5bc3747117e1.htm">2.2.2.8.1</a>).</p>
