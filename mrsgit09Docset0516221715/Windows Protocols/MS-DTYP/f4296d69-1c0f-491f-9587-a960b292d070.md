<p>An SDDL string is a single sequence of characters. The
format can be <a href="a66edeb1-52a0-4d64-a93b-2f5c833d7d92.html#gt_100cd8a6-5cb1-4895-9de6-e4a3c224a583">ANSI</a> or <a href="a66edeb1-52a0-4d64-a93b-2f5c833d7d92.html#gt_c305d0ab-8b94-461a-bd76-13b40cb8c4d8">Unicode</a>; the actual
protocol MUST specify the character set that is used. Regardless of the
character set used, the characters that can be used are alphanumeric and
punctuation.</p>

<p>The format for an SDDL string is described by the following
ABNF (as specified in <a href="https://go.microsoft.com/fwlink/?LinkId=123096">[RFC5234]</a>)
grammar, where the elements are as shown here.<a id="Appendix_A_Target_77"></a><a href="11e1608c-6169-4fbc-9c33-373fc9b224f4.html#Appendix_A_77" aria-label="Product behavior note 77">&lt;77&gt;</a></p>
