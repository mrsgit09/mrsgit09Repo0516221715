<p>The SERVER_INFO_100 structure contains information about the
specified server, including the name and platform.</p>

<dl>
<dd>
<div><pre> typedef struct _SERVER_INFO_100 {
   DWORD sv100_platform_id;
   [string] wchar_t* sv100_name;
 } SERVER_INFO_100,
  *PSERVER_INFO_100,
  *LPSERVER_INFO_100;
</pre></div>
</dd></dl>
