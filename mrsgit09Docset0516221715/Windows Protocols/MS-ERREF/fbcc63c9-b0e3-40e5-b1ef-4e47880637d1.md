<p>The Windows Error Codes document is a companion reference to
the protocol specifications. It documents the common usage details for those <a href="0642cb2f-2075-4469-918c-4441e69c548a.md">HRESULT values</a>, <a href="18d8fbe8-a967-4f1c-ae50-99ca8e491d2d.md">Win32 error codes</a>, and <a href="87fba13e-bf06-450e-83b1-9241dc81e781.md">NTSTATUS values</a> that are
referenced by specifications in the protocol documentation set.</p>

<p>Sections 1.7 and 2 of this specification are normative. All
other sections and examples in this specification are informative.</p>
