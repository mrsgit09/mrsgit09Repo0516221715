<p>The <b>StagingNameCheckResponse </b>element identifies
whether the specified staging name is unique or the standard operation result.</p>

element.</p>

     &lt;/xs:sequence&gt;
   &lt;/xs:complexType&gt;
 &lt;/xs:element&gt;
</pre></div>
</dd></dl>

<p><b>IsUnique</b>: Specifies whether the staging name
is unique.</p>

<p><b>OperationResult</b>: Specifies the list of errors
that is returned if the operation fails.</p>
