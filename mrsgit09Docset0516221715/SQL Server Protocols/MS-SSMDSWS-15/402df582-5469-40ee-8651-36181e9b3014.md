<p><b>AccessPermission</b>: Specifies the type of
access, if the user has been granted access permission.</p>

<p><b>AuditInfo</b>: Specifies the modification history
of a stored object.</p>

<p><b>Comment</b>: Specifies the text of an annotation.</p>

<p><b>Id</b>: Specifies the unique integer identifier of
the annotation.</p>

<p><b>RevisionId</b>: Specifies the unique identifier of
a revision item of the annotation.</p>

<p><b>SecurityPermission</b>: Specifies the
authorization of the user for the annotation. This property is always
read-only.</p>
