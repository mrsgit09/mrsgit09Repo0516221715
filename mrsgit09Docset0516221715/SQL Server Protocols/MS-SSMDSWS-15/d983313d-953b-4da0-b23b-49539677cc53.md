<p><b>EntityId</b>: Specifies the entity of the
parent-child data to be retrieved.</p>

<p><b>HierarchyId</b>: Specifies the hierarchy of the
parent-child data to be retrieved.</p>

<p><b>HierarchyType</b>: Specifies the hierarchy type.</p>

<p><b>Members</b>: Specifies the member data that is
retrieved by this operation.</p>

<p><b>ModelId</b>: Specifies the model of the
parent-child data to be retrieved.</p>

<p><b>VersionId</b>: Specifies the version of the
parent-child data to be retrieved.</p>
