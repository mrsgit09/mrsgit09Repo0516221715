<p><b>CreatedDateTime</b>: Specifies the date and time
at which the associated object was created.</p>

<p><b>CreatedUserId</b>: Specifies the <a href="95c9ef80-bda5-4be4-b5ee-3d261934a440.md">Identifier</a> of the user who
created the associated object.</p>

<p><b>CreatedVersionId</b>: Specifies the <b>Identifier</b>
of the version in which the associated object was created.</p>

<p><b>RevisionId</b>: Specifies the identifier of the
revision that is associated with the identified member.</p>

<p><b>UpdatedDateTime</b>: Specifies the date and time
at which the associated object was last updated.</p>

<p><b>UpdatedUserId</b>: Specifies the <b>Identifier</b>
of the user who last updated the object.</p>

<p><b>UpdatedVersionId</b>: Specifies the <b>Identifier</b>
of the version in which the associated object was last updated.</p>
