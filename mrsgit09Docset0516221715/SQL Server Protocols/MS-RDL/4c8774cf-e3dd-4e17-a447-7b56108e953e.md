<p><b><i>Applies to </i></b><a href="a7e2ad00-07c8-4f6d-80ab-3ad55df7b233.md"><b><i>RDL 2003/10</i></b></a><b>
<i>and </i></b><a href="3ebe2912-4958-4832-b391-cad1f5e13338.md"><b><i>RDL 2005/01</i></b></a></p>

<p>The <b>List.PageBreakAtEnd</b> element specifies that a
renderer inserts a page break at the end of a <a href="ea4c625c-0558-4fb3-b3b8-bde6c160b1e2.md">List</a>. This element is
optional. The value of <b>List.PageBreakAtEnd</b> MUST be a <a href="4802fa14-3619-43fa-9898-3acab160a24c.md">Boolean</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.2). If this element is not specified, its value is interpreted as false.</p>
