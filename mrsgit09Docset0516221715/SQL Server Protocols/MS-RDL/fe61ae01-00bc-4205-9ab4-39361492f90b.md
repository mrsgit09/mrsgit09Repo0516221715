<p>The <b>DataSet.Filters</b> element specifies a set of
filters to apply to each row in a <a href="a14782b0-2e2f-4305-83a3-3de3fd750b6a.md">DataSet</a>. This element is
optional and is of type <a href="4075354a-2747-4ce0-ba0f-3e32a950f605.md">Filters</a>.</p>
