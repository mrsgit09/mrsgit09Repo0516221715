<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.md"><b><i>RDL 2008/01</i></b></a><b><i>,
</i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.md"><b><i>RDL 2010/01</i></b></a><b><i>,
and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.md"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>GaugeLabel.Height</b> element specifies the height of
a <a href="3d97eae8-d903-471f-b151-9cc2bdbe91af.md">GaugeLabel</a> as a
percentage of <a href="cc3643bc-b12e-4458-974a-ffc2ba097c8d.md">GaugeLabel.ParentItem</a>.
If the peer element <b>GaugeLabel.ParentItem</b> is not specified, the value of
the <b>GaugeLabel.Height</b> element is interpreted as relative to the height
of the <a href="f01744d3-79fa-4f30-94bf-a1ffa6bde2ac.md">GaugePanel</a>.</p>

<p>The <b>GaugeLabel.Height</b> element is optional. If this
element is present, its value MUST be a <a href="c7d0946f-992e-4abc-a304-09b53e030692.md">Float</a> (<a href="https://go.microsoft.com/fwlink/?LinkId=90610">[XMLSCHEMA2]</a> section
3.2.4) or an expression that evaluates to a <b>Float</b>. If the <a href="f9d6ec8d-393e-41b9-9ba6-e13c09aff56c.md">GaugePanel.AutoLayout</a>
property for the <b>GaugeLabel.Height</b> element’s parent <b>GaugePanel</b>
element is set to true, the <b>GaugeLabel.Height</b> element is ignored. If
this element is not present, its value is interpreted as 0.</p>

element.</p>
