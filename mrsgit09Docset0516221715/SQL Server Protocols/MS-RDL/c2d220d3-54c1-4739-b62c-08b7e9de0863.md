<p><b><i>Applies to </i></b><a href="3428e690-a348-4ec7-8a6a-8efb42d2cdee.md"><b><i>RDL 2010/01</i></b></a><b><i>

<p>The <b>MapColorScale.MapSize</b> element specifies the size
of a <a href="fc14b477-a2d2-4048-843d-6a19beeb30bf.md">MapColorScale</a>.
This element is optional. If this element is not present, the color scale size
MUST be automatically calculated. This element is of type <a href="d34e1a7e-ada4-4989-9c0f-fbb69c9347ec.md">MapSize</a>.</p>
