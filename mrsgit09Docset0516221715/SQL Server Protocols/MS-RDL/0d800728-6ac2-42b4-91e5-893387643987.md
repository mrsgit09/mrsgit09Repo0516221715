<p>The <b>CustomData.DataColumnGroupings</b> element specifies
the data column grouping hierarchy for a <a href="6bb7b35c-e517-4444-a96b-9f2ccdd1a642.md">CustomReportItem</a>. This
element MUST be specified exactly once and is of type <a href="2eb7281a-4d84-4c97-ad39-89dac1dbc1bc.md">DataColumnGroupings</a>.</p>
