collection of <a href="1d8a9691-b173-4e24-9ea9-1f486bc824fd.md">TablixMember</a>

NOT be specified more than once. If this element is specified, it is of type <a href="1531211e-bbb1-4ef0-b5a4-d8e4c08a6e4c.md">TablixMembers</a>.</p>
