<p>The <b>Style.PaddingTop</b> element specifies the padding
between the top edge of a report item and its contents. This element is
optional. If this element is present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.md">RdlSize</a> or an expression
that evaluates to an <b>RdlSize</b>.</p>

<p>The value of the <b>Style.PaddingTop</b> element MUST NOT be
a size that is greater than 1000 pt or the equivalent. If the grandparent
element of the <b>Style.PaddingTop</b> element is not <a href="469d0032-b5ec-43d9-ab36-d3a88b9cc1f6.md">Textbox</a>, <a href="44172a0a-a53f-423e-be81-08352a109961.md">Subtotal</a>, or <a href="63e1e5ab-7c49-4f62-8dbd-62d85de2b153.md">Image</a>, the <b>Style.PaddingTop</b>
element is ignored.</p>
