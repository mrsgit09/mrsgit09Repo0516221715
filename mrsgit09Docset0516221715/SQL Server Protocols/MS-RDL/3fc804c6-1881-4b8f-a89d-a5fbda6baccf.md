<p>The <b>DynamicColumns.Visibility</b> element specifies the
presentational presence of the <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_d8aee082-104c-4091-9a95-acfb945b489b">dynamic columns</a> for a
grouping in a <a href="25419c0a-c7c6-43d7-8ca5-1af842666dcb.md">Matrix</a>.
This element is optional and is of type <a href="9505fbda-7f65-4874-a54a-1944059812e0.md">Visibility</a>.</p>
