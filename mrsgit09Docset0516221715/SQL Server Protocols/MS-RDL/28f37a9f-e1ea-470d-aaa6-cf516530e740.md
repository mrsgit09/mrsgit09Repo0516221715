by which to sort the detail data in a <a href="660db744-699e-4ca3-a2d6-a5cab4bcf9b0.md">Table</a>. This element is
optional and is of type <a href="1bf282fa-97a7-4dc0-b8af-171f4e777f73.md">Sorting</a>.
</p>
