            <h1 class="heading">6.123 rsMissingParameterDefault</h1>
        </div>
        <div id="mainSection">
            <div id="mainBody">
                <div id="allHistory" class="saveHistory"></div>
                <div id="sectionSection0" class="section" name="collapseableSection">
                    

<p><i>rsMissingParameterDefault</i> is a critical error that
will occur when a <a href="8e66448d-9239-490c-8c81-5d4bce32e4d8.md">ReportParameter.DefaultValue</a>
element is not present for a <a href="7c3f4c83-9172-48db-94c1-693295c5d623.md">ReportParameter</a>
element that is either missing a <a href="d75f59d9-f428-4464-a6c0-8978b0e025e2.md">ReportParameter.Prompt</a>
child element and has <a href="aeb93aab-9673-4c7a-998a-1f6391d7accb.md">ReportParameter.Nullable</a>
set to false, or none of its <a href="c4eaa375-0403-4ab5-bd3d-f9fd818675f8.md">ValidValues.ParameterValues</a>
elements within <a href="241ed24f-ce24-46dd-963a-734fdba1532c.md">ValidValues</a>
is set to NULL.</p>
