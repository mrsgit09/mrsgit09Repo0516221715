<p>The <b>Page.TopMargin</b> element specifies the height of
the top margin of a <a href="b5e525d5-00d6-4e1a-8813-55f327da6b4c.md">Page</a>.
The <b>Page.TopMargin</b> element is optional. If this element is present, its
value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.md">RdlSize</a>.
If this element is not present, its value is interpreted as 0in.</p>
