center of a <a href="55679f1a-a5b6-4b08-b284-ff6e27deedb4.md">MapViewport</a>

element is optional and is of type <a href="21154ffd-ecd3-4e2f-ae18-f056a5350467.md">MapCustomView</a>.</p>
