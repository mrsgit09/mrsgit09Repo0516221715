<p>The <b>PointerImage.OffsetY</b> element specifies the Y, or
vertical, offset for a <a href="ecf5073e-d4ae-4742-a92f-6790140b0fe6.md">PointerImage</a>.
This element is optional. If this element is present, its value MUST be an <a href="b40c092e-4fe5-4f7b-a0bf-c98df1361c90.md">RdlSize</a>.</p>
