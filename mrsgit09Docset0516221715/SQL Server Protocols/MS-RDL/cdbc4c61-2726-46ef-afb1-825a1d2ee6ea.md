<p><b><i>Applies to </i></b><a href="1e855f94-4617-47e4-b89e-0856c6cb420f.md"><b><i>RDL 2008/01</i></b></a><b><i>,

and </i></b><a href="52ce3983-2bfc-4e72-9359-42aaf5fe4509.md"><b><i>RDL 2016/01</i></b></a></p>

<p>The <b>GaugePanel.Name</b> attribute specifies a unique
identifier for a <a href="f01744d3-79fa-4f30-94bf-a1ffa6bde2ac.md">GaugePanel</a>.
The <b>GaugePanel.Name</b> attribute MUST be specified. The value of this
attribute MUST be a case-sensitive <a href="b2482b3f-74ab-4ca8-a9e5-c07955011743.md#gt_cb2ad790-a668-429f-84fa-f3dd67517e9b">CLS-compliant identifier</a> <a href="https://go.microsoft.com/fwlink/?LinkId=147989">[UTR15]</a> that is
unique for all the report items within a <a href="6bbaafec-020b-406c-b4e7-5e4318b616cb.md">Report</a>.</p>
