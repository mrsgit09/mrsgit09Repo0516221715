<p>The <b>Tablix.Style</b> element specifies the <a href="ea446209-9c6a-46ce-b472-fae8b8350b37.md">Style</a> element of a <a href="e42fb86e-799a-4202-8845-ac38831efccb.md">Tablix</a>. The <b>Tablix.Style</b>
element is optional and MUST NOT be specified more than once. If this element
is specified, it is of type <b>Style</b>.</p>

<p>See also <a href="8daadd7d-8f1e-409d-b036-4780a4b173e0.md">Tablix.OmitBorderOnPageBreak</a>.</p>
