<p>The <b>BackgroundColor</b> property specifies the color to
use for the background. If this property is specified, it MUST be part of a <a href="04bf25a1-2f43-4acf-b9eb-b9fa2dc45202.html">Style</a> record. If the <b>BackgroundColor</b>
property is not specified, the background color is interpreted as transparent.</p>
