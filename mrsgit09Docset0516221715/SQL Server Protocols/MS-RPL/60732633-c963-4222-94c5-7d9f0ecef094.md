            <h1 class="heading">2.3.68 BorderWidthLeft</h1>
        </div>

<p>The <b>BorderWidthLeft</b> property specifies the width of
the left border. If this property is specified, it MUST be part of a <a href="04bf25a1-2f43-4acf-b9eb-b9fa2dc45202.md">Style</a> record. The value of
the <b>BorderWidthLeft</b> property overrides the <a href="b1a40301-fa1f-4d34-b0e5-34b5e50dca71.md">BorderWidth</a> property for
the left border. If the <b>BorderWidthLeft</b> property is not defined, the
value of <b>BorderWidth</b> is used.           </p>
