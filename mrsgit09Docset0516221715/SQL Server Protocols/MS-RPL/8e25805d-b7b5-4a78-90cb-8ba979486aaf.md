            <h1 class="heading">2.3.102 CellItemOffset</h1>
        </div>
        <div id="mainSection">
            <div id="mainBody">
                <div id="allHistory" class="saveHistory"></div>
                <div id="sectionSection0" class="section" name="collapseableSection">
                    

<p>The <b>CellItemOffset</b> property specifies the position of
a <a href="75f1a870-2f17-4806-b286-e67c7239e103.html">ReportElementEnd</a>
record in an RPL stream. The value of the <b>CellItemOffset</b> property MUST
correspond to the top-level <a href="75ae48f7-746b-4b41-919c-6699fa28b3ef.html#gt_c6f8e999-fca9-4e79-96e7-fb4c2c43d601">report
item</a> of a <a href="75ae48f7-746b-4b41-919c-6699fa28b3ef.html#gt_f9f5d4be-2a9e-4556-90f6-d4ed1678f0b4">tablix</a>
cell.</p>

<p>If the <b>CellItemOffset</b> property is specified, it MUST
be part of a <a href="20e3b37d-978d-467f-b068-d7a2746e37da.html">TablixCorner</a>,
<a href="0d5c4157-00d0-4268-854f-f274a9d102fb.html">TablixRowHeader</a>, <a href="968a6852-ede1-4bf1-8006-1dab2aea178b.html">TablixColumnHeader</a>, or <a href="fa12273f-80a1-432a-bced-a765ff87dbc7.html">TablixBodyCell</a> record.</p>

specifies the start of the <b>CellItemOffset</b> property. The value of the <b>cellItemOffsetStart</b>
field MUST be the following value.</p>

<p><b>cellItemOffsetValue (8 bytes): </b>A signed
integer that specifies the position of a <b>ReportElementEnd</b> record in the
RPL stream. The value of this field MUST correspond to the top-level <a href="422387f7-880f-4d86-9e88-2a5d2e8f191e.html">ReportItem</a> record of the
parent tablix cell.</p>
