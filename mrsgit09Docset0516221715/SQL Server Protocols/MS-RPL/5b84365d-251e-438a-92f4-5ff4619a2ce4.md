<p>The <b>ListLevel</b> property specifies the numbering style
and/or indentation level of a <a href="3024abc3-23db-494b-a63a-6bd565e4500b.html">Paragraph</a> record. If the <b>ListLevel</b>
property is specified, it MUST be part of a <a href="d7f6cef2-01c6-4562-a4a0-5f205d79963e.html">Paragraph.ElementProperties</a>
record. If the <b>ListLevel</b> property is not specified, the default value is
used. </p>

field MUST be the following value.</p>
