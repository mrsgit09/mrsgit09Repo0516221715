of a report. If this property is specified, it MUST be part of a <a href="a9b28610-5438-470d-84bb-0608d07ddc46.md">ReportProperties</a>
record.           </p>

<p><b>authorStart (1 byte): </b>A byte that specifies
the start of the <b>Author</b> property. The value of the <b>authorStart</b>
field MUST be the following value.</p>
