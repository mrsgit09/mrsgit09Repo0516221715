<p>The <b>LineHeight</b> property specifies the height of a
contiguous set of characters within a <a href="75ae48f7-746b-4b41-919c-6699fa28b3ef.md#gt_c6f8e999-fca9-4e79-96e7-fb4c2c43d601">report item</a>. If this
property is specified, it MUST be part of a <a href="04bf25a1-2f43-4acf-b9eb-b9fa2dc45202.md">Style</a> record.           </p>

<p><b>lineHeightStart (1 byte): </b>A byte that
specifies the start of the <b>LineHeight</b> property. The value of the <b>lineHeightStart</b>
field MUST be the following value.</p>
