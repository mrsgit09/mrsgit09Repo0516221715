<p>The <b>PaddingLeft</b> property specifies the size of the
padding to apply to the left of the contents. If the <b>PaddingLeft</b> property
is specified, it MUST be part of a <a href="04bf25a1-2f43-4acf-b9eb-b9fa2dc45202.html">Style</a> record. If the <b>PaddingLeft</b>
property is not specified, the default value is used.</p>
