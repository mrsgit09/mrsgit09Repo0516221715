<p><b>shared (1 byte): </b>A byte that specifies the
start of the <b>SharedStyleProperties</b> record. The value of the <b>shared</b>
field MUST be 0x00.</p>

<p><b>styleProperties (variable): </b>A collection of
structures that specifies style properties that are shared by multiple records.
The order of the structures in the collection is arbitrary. The collection can
be empty.</p>

<p><b>delimiter (1 byte): </b>A byte that specifies the
end of the <b>SharedStyleProperties</b> record. The value of the <b>delimiter</b>
field MUST be 0xFF.</p>
