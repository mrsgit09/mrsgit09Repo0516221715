<p>The <b>Calendar</b> property specifies the calendar to use
for formatting dates. If the <b>Calendar</b> property is specified, its value
MUST be compatible with the Microsoft .NET Framework Language setting. The
property MUST be part of a <a href="04bf25a1-2f43-4acf-b9eb-b9fa2dc45202.html">Style</a>
record. If the <b>Calendar</b> property is not specified, the default value is
used.</p>

<p><b>calendarValue (1 byte): </b>A byte that specifies
the calendar to use for formatting dates. The value of the <b>calendarValue</b>
field is of type <a href="9ab5c405-b697-4574-9a36-c6e5c070642e.html">Calendars</a>
and it MUST be contained in the <b>Calendars</b> <a href="75ae48f7-746b-4b41-919c-6699fa28b3ef.html#gt_846463b5-421c-4d6b-8d82-79d44db666fa">enumeration</a>.<a id="Appendix_A_Target_57"></a><a href="1d022514-2a2f-41df-b2f8-36f19e474fa5.html#Appendix_A_57" aria-label="Product behavior note 57">&lt;57&gt;</a> The default value is defined by
default calendar of the <a href="235d01b9-ca4b-4e7b-ba6d-646e5d859649.html">Language</a>
peer style property.</p>
