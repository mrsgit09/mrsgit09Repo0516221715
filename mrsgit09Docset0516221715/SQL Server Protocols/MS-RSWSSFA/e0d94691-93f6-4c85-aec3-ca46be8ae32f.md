<p>The ReportServiceAuthentication web service operates against
a URL. The URL is identified by protocol clients. The protocol server endpoint
is formed by appending &quot;/ReportServiceAuthentication.asmx&quot; to the
URL, as in the following example:</p>

<p>http://www.contoso.com/ReportSite/_vti_bin/ReportServer/ReportServiceAuthentication.asmx</p>

<p>It is assumed that authentication has been performed by the
underlying protocol. The account that accesses this web service is assigned
with proper permissions to the site and its items.</p>
