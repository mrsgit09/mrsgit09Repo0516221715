   INT32(100);
   *CUBE;
   CLOSE
</pre></div>
</dd></dl>

 <tr>

  </td>
 </tr>
 <tr>
  <td>

  </td>

 </tr>
 <tr>
  <td>

  </td>

  <p>An array of data structures that contain information

  </td>
 </tr>
 <tr>
  <td>
