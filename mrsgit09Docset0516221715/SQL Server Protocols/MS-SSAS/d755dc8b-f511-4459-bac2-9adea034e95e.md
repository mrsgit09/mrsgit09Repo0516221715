<p>The <b>DMSCHEMA_MINING_FUNCTIONS</b> rowset contains the
following columns.</p>

<table>
 <thead>
  <tr>
   <th>
   <p>Name</p>
   </th>
   <th>
   <p>Type</p>
   </th>
   <th>
   <p>Restriction</p>
   </th>
   <th>
   <p>Description</p>
   </th>
  </tr>
 </thead>
 <tr>
  <td>
  <p>SERVICE_NAME</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p>Yes</p>
  </td>
  <td>
  <p>The name of the algorithm.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>FUNCTION_NAME</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p>Yes</p>
  </td>
  <td>
  <p>The name of the function.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>FUNCTION_SIGNATURE</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>The signature of the function.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>RETURNS_TABLE</p>
  </td>
  <td>
  <p>xsd:boolean</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>When true, indicates that the function returns a table
  (such as a histogram table). When false, indicates that the function returns
  scalar content (such as the length of the character argument).</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>DESCRIPTION</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>A human-readable description of the function.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>HELP_FILE</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>The name of the file that contains this function's
  documentation.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>HELP_CONTEXT</p>
  </td>
  <td>
  <p>xsd:int</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>The help context ID for this function.</p>
  </td>
 </tr>
</table>
