<p>The <b>MDSCHEMA_SETS</b> rowset contains the following
columns.</p>

<table>
 <thead>
  <tr>
   <th>
   <p>Name</p>
   </th>
   <th>
   <p>Type</p>
   </th>
   <th>
   <p>Restriction</p>
   </th>
   <th>
   <p>Description</p>
   </th>
  </tr>
 </thead>
 <tr>
  <td>
  <p>CATALOG_NAME</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p>Yes</p>
  </td>
  <td>
  <p>The name of the database.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>SCHEMA_NAME</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p>Yes</p>
  </td>
  <td>
  <p>The name of the schema.<a id="Appendix_A_Target_223"></a><a href="b9ac4859-2662-44ca-b131-9addd8b953dc.html#Appendix_A_223" aria-label="Product behavior note 223">&lt;223&gt;</a></p>
  </td>
 </tr>
 <tr>
  <td>
  <p>CUBE_NAME</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p>Yes</p>
  </td>
  <td>
  <p>The name of the cube.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>SET_NAME</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p>Yes</p>
  </td>
  <td>
  <p>The name of the set, as specified in the CREATE SET statement.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>SCOPE</p>
  </td>
  <td>
  <p>xsd:int</p>
  </td>
  <td>
  <p>Yes</p>
  </td>
  <td>
  <p>The scope of the set. The set can be a session-defined
  set or a global-defined set.</p>
  <p>This column can have one of the following values:</p>
  <ul><li><p><span><span>  
  </span></span><span>1 - Global</span></p>
  </li><li><p><span><span>  
  </span></span><span>2 – Session</span></p>
  </li></ul></td>
 </tr>
 <tr>
  <td>
  <p>DESCRIPTION</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>A description of the set.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>EXPRESSION</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>The expression for the set.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>DIMENSIONS</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>A comma-delimited list of hierarchies included in the
  set.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>SET_CAPTION</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>A caption associated with the set.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>SET_DISPLAY_FOLDER</p>
  </td>
  <td>
  <p>xsd:string</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>The display folder.</p>
  </td>
 </tr>
 <tr>
  <td>
  <p>SET_EVALUATION_CONTEXT</p>
  </td>
  <td>
  <p>xsd:int</p>
  </td>
  <td>
  <p> </p>
  </td>
  <td>
  <p>The context for the set. The set can be static or
  dynamic.</p>
  <p>This column can have one of the following values:</p>
  <ul><li><p><span><span>  
  </span></span><span>1 - STATIC</span></p>
  </li><li><p><span><span>  
  </span></span><span>2 – DYNAMIC</span></p>
  </li></ul></td>
 </tr>
</table>
